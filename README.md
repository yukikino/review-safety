# 低評価口コミ・クレーム対応マニュアル

悪い口コミやクレーム対応に困っている店舗オーナー向けの「返信テンプレ＋その後の守り方」実務メディアです。

## サイトの目的

このサイトは、「Googleマップで★1レビューを書かれた」「ホットペッパーで厳しいクレームがあった」という悩みを持つ店舗オーナー・店長に向けて、実務で使える返信テンプレートと対応フローを提供します。

### 想定読者

- 飲食店、エステ、マツエク、美容クリニック、パーソナルジム、整体院などのオーナー・店長
- 個人店舗〜小規模複数店舗の責任者
- Googleマップやホットペッパーなどで★1レビューや厳しいクレームを書かれて、どう返信すべきか困っている人

### 読者の悩みフェーズ

本サイトでは、読者の状況を2つのフェーズに分けてサポートします：

**フェーズ1: まだ角を立てずに丁寧に返信したい**
- 「待たされた」「接客が冷たい」など、ある程度事実ベースの不満レビュー
- 丁寧に謝りつつイメージを回復したい
- 公開返信でそのまま使える文章が欲しい

**フェーズ2: 事実と違う・誹謗に近い内容で精神的にギリギリ**
- 事実と異なること・誹謗に近いことを書かれた
- 線を引きたい / 削除依頼したい
- トラブル化していて法的対応も視野に入れている

## コンテンツ構成

### ディレクトリ構造

```
content/
├── mild-response/              # 通常の悪い口コミ対応
│   ├── waiting-complaint.md        # 「待たされた」への返信
│   ├── staff-attitude.md           # 「接客が冷たい」への返信
│   └── high-price-complaint.md     # 「高い割に効果ない」への返信
├── escalation/                 # 強め案件（事実と異なる等）
│   ├── false-claims.md             # 事実と違う口コミへの対応
│   └── defamation-handling.md      # 名誉毀損レベルの口コミ対応
├── industry-specific/          # 業種別テンプレ
│   ├── beauty-clinic.md            # 美容クリニック（薬機法対応）
│   └── restaurant.md               # 飲食店（食中毒・衛生対応）
└── playbook/                   # 全体フロー
    └── complete-workflow.md        # 対応フロー完全ガイド（ハブ記事）
```

### カテゴリ説明

#### 1. 通常対応（mild-response）
相手に角を立てずに謝罪しつつ、イメージを回復する返信文を紹介。
- 「待たされた」への返信テンプレ
- 「接客が冷たい」への返信テンプレ
- 「高い割に効果ない」への返信テンプレ

#### 2. 強め案件対応（escalation）
事実と異なる内容・名誉毀損レベルの口コミへの対応を解説。
- 公開返信でどこまで反論していいか
- 削除依頼の手順と判断基準
- 法的手続きの流れ

#### 3. 業種別対応（industry-specific）
業種ごとに絶対に言ってはいけない表現・信頼される表現を差し込む。
- 美容クリニック（薬機法・医療広告ガイドライン対応）
- エステ・痩身サロン
- パーソナルジム
- 飲食店（食中毒・衛生対応）

#### 4. 全体フロー（playbook）
店舗スタッフ向けの「クレームがあったらどう記録してオーナーに共有し、どのタイミングで返信を出すか」をまとめた完全ガイド。

## 記事の作成方法

### テンプレートを使う

`templates/review-response.md`をコピーして、新しい記事を作成してください。

```bash
cp templates/review-response.md content/mild-response/new-article.md
```

### Frontmatterの設定

```yaml
---
title: "記事タイトル（検索キーワードを含める）"
description: "記事の説明文（120〜160文字）"
date: "2025-11-03"
author: "編集部"
category: "mild-response"  # mild-response, escalation, industry-specific, playbook
tags: ["口コミ対応", "クレーム返信"]
cta: "meo"  # meo, reputation, mixed
---
```

### CTAの種類

- **meo**: MEO/口コミ管理ツールの無料トライアル向け（mild-response記事用）
- **reputation**: 風評対策・評判管理コンサルの無料相談向け（escalation記事用）
- **mixed**: 両方のCTAを含む（playbook記事用）
