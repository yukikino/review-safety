## この記事は誰のため？

この記事は、学習塾・予備校・個別指導塾の経営者で、Googleマップや口コミサイトで「成績が上がらなかった」「合格実績が嘘」「高額な教材を売りつけられた」と書かれてしまった方に向けて書かれています。

教育業界の口コミは、保護者の期待値が高く、成果が出ないと低評価につながります。また、合格実績の誇大広告や高額教材の押し売りは、景品表示法・特定商取引法違反のリスクがあります。

この記事を読むことで、教育業界特有のクレームへの返信テンプレート、法令遵守の広告表現、トラブル予防策が分かります。

## 学習塾・予備校特有の口コミトラブル

### トラブル1: 成績が上がらない

**よくあるクレーム**:
- 「半年通ったのに、偏差値が1も上がらない」
- 「個別指導なのに、成績が下がった」
- 「授業料が高いのに、効果なし」

**原因**:
- 生徒の学習意欲・能力の個人差
- 塾の指導方法と生徒の相性が悪い
- 保護者の過度な期待

### トラブル2: 合格実績の誇大広告

**よくあるクレーム**:
- 「『東大合格者100名』と書いてあったが、実際は他校舎も含めた数字」
- 「合格実績に名前を使われた（本人の許可なく）」
- 「体験授業だけの生徒も『合格者』にカウントしている」

**法的リスク**:
- 景品表示法違反（優良誤認表示）
- 不正競争防止法違反（誤認惹起行為）

### トラブル3: 高額教材の押し売り

**よくあるクレーム**:
- 「入塾時に30万円の教材を買わされた」
- 「断りにくい雰囲気で契約させられた」
- 「クーリングオフができないと言われた」

**法的リスク**:
- 特定商取引法違反（不実告知、威迫行為）
- 消費者契約法違反

### トラブル4: 講師の質・指導力不足

**よくあるクレーム**:
- 「学生アルバイトばかりで、プロ講師がいない」
- 「講師がコロコロ変わる」
- 「授業中、講師がスマホを見ている」

**原因**:
- 講師の採用基準が低い
- 研修不足
- 人手不足による講師の頻繁な交代

### トラブル5: 退塾時のトラブル

**よくあるクレーム**:
- 「退塾を申し出たら、引き止められた」
- 「退塾月の授業料を返金してくれない」
- 「教材費の返金を拒否された」

**法的リスク**:
- 特定商取引法違反（中途解約の妨害）
- 消費者契約法違反

## 公開返信テンプレート（学習塾・予備校用）

### テンプレート1: 成績が上がらない

```
この度はご入塾いただき、誠にありがとうございました。

成績向上につきまして、ご期待に添えず申し訳ございませんでした。

学習効果には個人差がございますが、お子様の学習状況につきまして、詳しくお話をお伺いしたく、恐れ入りますが教室までご連絡いただけますでしょうか。

今後の学習プランについて、ご相談させていただきます。

[塾名] 教室長 〇〇
```

**ポイント**:
- 「成績が上がる」と断定しない（景品表示法違反のリスク）
- 個人差があることを明記
- 非公開での相談を促す

### テンプレート2: 合格実績の誇大広告疑惑

```
この度はお問い合わせいただき、ありがとうございます。

合格実績につきまして、誤解を招く表現があったこと、お詫び申し上げます。

当塾の合格実績は、全校舎の合計人数を掲載しております。各校舎の実績は、ホームページに詳細を記載しておりますので、ご確認いただけますと幸いです。

今後、より分かりやすい表記を心がけてまいります。

[塾名] 教室長 〇〇
```

**ポイント**:
- 合格実績の集計方法を明記
- 誤解を招く表現を認め、改善を約束
- 詳細情報の掲載場所を案内

### テンプレート3: 高額教材の押し売り

```
この度はご入塾いただき、ありがとうございました。

教材のご案内につきまして、ご不快な思いをさせてしまい申し訳ございませんでした。

当塾では、無理な勧誘は行わないよう徹底しておりますが、スタッフの説明が不十分であった可能性がございます。

今後、お客様のご意向を最優先に、丁寧な説明を心がけてまいります。

[塾名] 教室長 〇〇
```

**ポイント**:
- 無理な勧誘をしていないことを明記
- スタッフ教育の徹底を約束
- 顧客の意向を尊重する姿勢を示す

### テンプレート4: 講師の質・指導力不足

```
この度はご入塾いただき、誠にありがとうございました。

講師の指導につきまして、ご不満な点があったこと、大変申し訳ございませんでした。

当塾では、講師の研修を定期的に実施し、指導力向上に努めております。

お子様の学習状況につきまして、詳しくお話をお伺いしたく、恐れ入りますが教室までご連絡いただけますでしょうか。

[塾名] 教室長 〇〇
```

**ポイント**:
- 講師研修の実施を明記
- 改善への取り組みを示す
- 個別の相談を促す

## 絶対に書いてはいけないNG返信表現

教育業界の返信には、炎上リスクの高いNG表現があります。

### NG例1: 成績向上を断定する

```
NG返信: 「当塾に通えば、必ず成績が上がります」
```

**リスク**:
- 景品表示法違反（優良誤認表示）
- 効果が出なかった場合、「詐欺だ」と批判される

**正しい表現**:
```
OK返信: 「効果には個人差がございます」
```

### NG例2: 生徒・保護者を批判する

```
NG返信: 「お子様のやる気がないのが原因です」
```

**リスク**:
- 保護者を激怒させる
- 「生徒のせいにするな」と炎上

**正しい表現**:
```
OK返信: 「学習効果には個人差がございます。お子様に合った学習プランをご提案します」
```

### NG例3: 他塾を批判する

```
NG返信: 「他の塾は学生バイトばかりですが、当塾はプロ講師です」
```

**リスク**:
- 不正競争防止法違反（誹謗中傷）
- 他塾から訴えられる可能性

**正しい表現**:
```
OK返信: 「当塾の講師は、全員が研修を受けております」
```

### NG例4: 合格実績を盛る

```
NG返信: 「当塾から東大合格者100名！」（実際は他校舎も含めた数字）
```

**リスク**:
- 景品表示法違反（優良誤認表示）
- 消費者庁から措置命令

**正しい表現**:
```
OK返信: 「当塾（全校舎合計）から東大合格者100名」
```

## 法令遵守のための広告表現ガイド

### 景品表示法で禁止されている表現

| 禁止表現 | 理由 | 代替表現 |
|---------|------|---------|
| 必ず成績が上がる | 効果の断定（優良誤認） | 効果には個人差があります |
| 100%合格保証 | 効果の断定（優良誤認） | 合格実績〇名（過去3年間） |
| 地域No.1 | 根拠のない表現 | お客様満足度〇%（自社調査） |
| 他塾より安い | 根拠のない比較 | 月額〇円（他塾と比較してご検討ください） |

### 特定商取引法の遵守（高額教材販売）

**特定継続的役務提供に該当する場合**:
- 期間: 2ヶ月を超える
- 金額: 5万円を超える

**遵守すべき内容**:
1. 契約書面の交付
2. クーリングオフ（8日間）の説明
3. 中途解約権の説明
4. 誇大広告の禁止

**違反した場合の罰則**:
- 業務停止命令
- 100万円以下の罰金

## 予防策: 教育トラブルを減らすために

### 予防策1: 入塾時のカウンセリング徹底

**カウンセリングで確認すべき内容**:
- 現在の学力（模試の偏差値、学校の成績等）
- 目標（志望校、目標偏差値等）
- 学習習慣（自宅学習時間、得意・苦手科目等）
- 保護者の期待値

**過度な期待を下げる**:
- 「偏差値を10上げるには、最低6ヶ月かかります」
- 「効果には個人差があります」
- 「塾だけでなく、自宅学習も重要です」

### 予防策2: 合格実績の適正表示

**適正な合格実績の表示方法**:
- 全校舎合計の場合、「全校舎合計」と明記
- 校舎ごとの実績も掲載
- 集計期間を明記（「過去3年間」等）
- 体験授業のみの生徒は除外

**NG表示**:
- 他校舎の実績を、自校舎の実績のように表示
- 体験授業のみの生徒を「合格者」にカウント
- 実績を盛る（10名を100名と表示）

### 予防策3: 教材費の透明化

**料金表の明示**:
- ホームページに料金表を掲載
- 入塾案内に教材費を明記
- 追加費用（夏期講習、模試代等）も説明

**強制ではないことを明記**:
- 「教材は任意購入です」
- 「ご自宅の教材を使っていただいても構いません」

**クーリングオフの説明**:
- 契約書に「8日以内ならクーリングオフ可能」と明記
- 口頭でも説明

### 予防策4: 講師の質向上

**講師採用基準の明確化**:
- 学歴（大学名、学部等）
- 指導経験（年数、合格実績等）
- 人柄（生徒とのコミュニケーション能力等）

**定期研修の実施**:
- 月1回の研修会
- 授業見学・フィードバック
- 模擬授業

**講師情報の公開**:
- ホームページに講師プロフィールを掲載
- 学歴、指導歴、合格実績を明記

### 予防策5: 定期的な面談・報告

保護者との定期面談を実施し、学習状況を報告します。

**面談の頻度**:
- 入塾時: 1回
- 3ヶ月ごと: 1回
- 受験前: 随時

**面談で報告すべき内容**:
- 学習状況（出席率、授業態度、小テスト結果等）
- 成績の推移（模試の偏差値、学校の成績等）
- 今後の学習プラン
- 志望校の合格可能性

**効果**:
- 保護者の不安を解消
- 「放置されている」というクレームを防ぐ
- 成績が上がらない場合も、早期に対策を打てる

## 学習塾・予備校の判例・事例

### 事例1: 合格実績の虚偽表示で措置命令

**概要**:
- 学習塾が「東大合格者100名」と広告
- 実際は、体験授業のみの生徒も含めた数字
- 消費者庁が景品表示法違反（優良誤認表示）として措置命令

**教訓**: 合格実績は正確に表示する。

### 事例2: 高額教材の押し売りで業務停止命令

**概要**:
- 学習塾が、入塾時に50万円の教材を強制販売
- クーリングオフの説明をしなかった
- 経済産業省が特定商取引法違反として業務停止命令（3ヶ月）

**教訓**: クーリングオフの説明を必ず行う。

### 事例3: 成績向上効果なしで返金請求（請求棄却）

**概要**:
- 保護者が「1年通ったのに成績が上がらなかった。授業料を返金しろ」として提訴
- 塾側が「効果には個人差がある」と反論
- 裁判所が「成績向上を保証していない」として請求棄却

**教訓**: 「必ず成績が上がる」と約束しない。
