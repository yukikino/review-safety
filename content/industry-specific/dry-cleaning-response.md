## この記事は誰のため？

この記事は、クリーニング店の経営者で、Googleマップや口コミサイトで「服が破れて返ってきた」「高級コートを紛失された」「色落ちした」と書かれてしまった方に向けて書かれています。

クリーニング業界の口コミは、衣類のトラブルが多く、賠償問題に発展しやすいのが特徴です。適切に対応しないと、高額な賠償請求や悪評拡散のリスクがあります。

この記事を読むことで、クリーニング事故への返信テンプレート、賠償基準の適用方法、トラブル予防策が分かります。

## クリーニング店特有の口コミトラブル

### トラブル1: 衣類の破損

**よくあるクレーム**:
- 「ワイシャツのボタンが取れていた」
- 「ダウンジャケットが破れていた」
- 「ニットが縮んだ」

**原因**:
- クリーニング工程でのミス（洗濯方法の誤り、乾燥温度の誤り等）
- 衣類の経年劣化
- 元々の欠陥（ボタンの糸が緩んでいた等）

### トラブル2: 衣類の紛失

**よくあるクレーム**:
- 「預けたコートが返ってこない」
- 「他の客の服と間違えて渡された」
- 「紛失したのに、謝罪がない」

**原因**:
- タグ付けミス
- 保管場所の混乱
- スタッフの確認不足

### トラブル3: 色落ち・変色

**よくあるクレーム**:
- 「黒いスカートが色落ちして茶色になった」
- 「白いシャツが黄ばんだ」
- 「色移りした」

**原因**:
- 漂白剤の使いすぎ
- 洗濯温度の誤り
- 他の衣類との混洗

### トラブル4: シミが取れない

**よくあるクレーム**:
- 「シミ抜きを頼んだのに、取れていない」
- 「シミ抜き代を返金してほしい」

**原因**:
- シミの種類によっては、完全除去が困難
- 時間が経過したシミは取れにくい

### トラブル5: 高額な賠償請求

**よくあるクレーム**:
- 「高級ブランドのコートを破損された。購入金額30万円を賠償しろ」
- 「クリーニング賠償基準を適用されたが、納得できない」

**法的問題**:
- クリーニング事故賠償基準の適用
- 購入金額と時価の差

## 公開返信テンプレート（クリーニング店用）

### テンプレート1: 衣類の破損

```
この度はお預かりした衣類につきまして、破損があり誠に申し訳ございませんでした。

ご指摘の件につきまして、詳しく確認させていただきたく、恐れ入りますが店舗までお越しいただけますでしょうか。

クリーニング事故賠償基準に基づき、適切に対応させていただきます。

[店舗名] 店長 〇〇
```

**ポイント**:
- 謝罪を明確に
- 「賠償基準に基づく」と明記（過度な賠償請求を防ぐ）
- 非公開での対応を促す

### テンプレート2: 衣類の紛失

```
この度はお預かりした衣類を紛失してしまい、誠に申し訳ございませんでした。

現在、店内を再度確認しておりますが、見つからない場合は、クリーニング事故賠償基準に基づき賠償させていただきます。

恐れ入りますが、店舗までご連絡いただけますでしょうか。

[店舗名] 店長 〇〇
```

**ポイント**:
- 紛失を認め、謝罪
- 賠償基準に基づくことを明記
- 再確認の努力を示す

### テンプレート3: 色落ち・変色

```
この度はお預かりした衣類に色落ちが発生し、誠に申し訳ございませんでした。

色落ちの原因につきまして、詳しく確認させていただきたく、恐れ入りますが店舗までお越しいただけますでしょうか。

クリーニング事故賠償基準に基づき、適切に対応させていただきます。

[店舗名] 店長 〇〇
```

**ポイント**:
- 謝罪を明確に
- 原因確認の姿勢を示す
- 賠償基準を明記

### テンプレート4: シミが取れない

```
この度はシミ抜きにつきまして、ご期待に添えず申し訳ございませんでした。

シミの種類や時間経過により、完全除去が困難な場合がございます。

再度シミ抜きを試みることも可能ですので、恐れ入りますが店舗までお越しいただけますでしょうか。

[店舗名] 店長 〇〇
```

**ポイント**:
- シミ除去の限界を説明
- 再チャレンジの提案
- 「シミ抜き代を返金しろ」という要求には、慎重に対応

## 絶対に書いてはいけないNG返信表現

クリーニング店の返信には、炎上リスクの高いNG表現があります。

### NG例1: 顧客の責任にする

```
NG返信: 「元々破れていたのでは？」
```

**リスク**:
- 「証拠もないのに客のせいにするな」と怒り、炎上
- 顧客を疑うと、信用を失う

**正しい対応**:
```
OK返信: 「詳しく確認させていただきたく、店舗までお越しください」
```

### NG例2: 賠償を拒否する

```
NG返信: 「経年劣化ですので、賠償はできません」
```

**リスク**:
- 一方的に賠償を拒否すると、「責任逃れだ」と批判される
- 訴訟リスク

**正しい対応**:
```
OK返信: 「賠償基準に基づき、適切に対応させていただきます」
```

### NG例3: 過度に賠償を約束する

```
NG返信: 「購入金額30万円を全額お支払いします」
```

**リスク**:
- クリーニング賠償基準では、時価で賠償（購入金額ではない）
- 過度な賠償は、他のクレームを誘発

**正しい対応**:
```
OK返信: 「賠償基準に基づき、時価で賠償させていただきます」
```

### NG例4: 公開の場で金額を議論する

```
NG返信: 「賠償額は5万円です」
```

**リスク**:
- 公開の場で金額を提示すると、議論が紛糾
- 「安すぎる」と批判される

**正しい対応**:
```
OK返信: 「詳細につきましては、店舗でご説明させていただきます」
```

## クリーニング事故賠償基準

### クリーニング事故賠償基準とは？

**制定機関**: 全国クリーニング生活衛生同業組合連合会（全ク連）

**法的性格**: 業界の自主基準（法律ではない）

**裁判での扱い**: 裁判所は、この基準を参考にして賠償額を判断することが多い

### 賠償額の計算方法

**基本式**:
```
賠償額 = 購入価格 × 補償割合
```

**補償割合**:
- 購入後の経過月数に応じて減少
- 平均使用年数（衣類の種類により異なる）を基準

**平均使用年数の例**:
- スーツ（夏物）: 2年
- スーツ（冬物）: 3年
- コート: 4年
- ワイシャツ・ブラウス: 2年
- セーター: 2年
- ダウンジャケット: 5年

**補償割合の計算式**:
```
補償割合 = (平均使用年数 - 経過年数) ÷ 平均使用年数 × 100%
```

**具体例**:
- 購入価格10万円のコート（平均使用年数4年）
- 購入後2年経過
- 補償割合 = (4 - 2) ÷ 4 × 100% = 50%
- 賠償額 = 10万円 × 50% = 5万円

### 賠償額の上限

**クリーニング代金の40倍**が上限

**具体例**:
- クリーニング代金: 2,000円
- 上限: 2,000円 × 40 = 8万円
- 購入価格30万円のコートでも、賠償額は最大8万円

### 賠償責任が免除されるケース

**以下の場合、賠償責任が免除される**:
1. 預かり証の引換期限を6ヶ月以上過ぎた場合
2. 事故発生後1年以内に申し出がなかった場合
3. 衣類の経年劣化による破損（クリーニングが原因ではない）
4. 顧客が「取扱い絵表示」を無視してクリーニングを依頼した場合

## 予防策: クリーニング事故を減らすために

### 予防策1: 受付時の確認徹底

**受付時に確認すべき内容**:
- 衣類の状態（破れ、ほつれ、シミ等）
- 取扱い絵表示の確認
- 特殊素材（シルク、カシミヤ等）の確認
- 付属品（ベルト、フード等）の確認

**受付票への記載**:
- 上記確認内容を記載
- 顧客の署名をもらう（「受付時の状態を確認した」という証拠）

### 予防策2: 衣類の写真撮影

受付時に、衣類の状態を写真撮影します。

**撮影すべき箇所**:
- 全体像
- シミ・汚れの箇所
- 破れ・ほつれの箇所
- 付属品（ベルト、ボタン等）

**効果**:
- 「元々破れていた」と証明できる
- トラブル時の証拠になる

### 予防策3: 取扱い絵表示の遵守

**取扱い絵表示とは**:
- 衣類のタグに記載された洗濯方法の表示
- 「水洗い不可」「漂白剤不可」「ドライクリーニング可」等

**遵守の重要性**:
- 表示を無視すると、破損・色落ちのリスク
- 表示通りにクリーニングすれば、賠償責任が軽減される

### 予防策4: 保管管理の徹底

**タグ付けの徹底**:
- 顧客ごとに異なるタグ番号
- タグが外れないように二重チェック

**保管場所の整理整頓**:
- 顧客ごとに分類
- 定期的な在庫確認

**効果**:
- 紛失リスクを減らす
- 「他の客の服と間違えた」というトラブルを防ぐ

### 予防策5: クリーニング保険への加入

クリーニング事故に備え、損害保険に加入します。

**保険の種類**:
- クリーニング賠償責任保険
- 受託者賠償責任保険

**保険金額の目安**:
- 1事故あたり100万円〜1,000万円

**効果**:
- 高額賠償に備えられる
- 「保険に加入している」ことを店内に掲示し、信用を得る

## クリーニング業界の判例・事例

### 事例1: ダウンジャケット破損で賠償請求（5万円）

**概要**:
- クリーニング店が、ダウンジャケットを破損
- 顧客が「購入価格20万円を賠償しろ」と請求
- クリーニング店が「賠償基準に基づき5万円」と回答
- 裁判所が「賠償基準は妥当」として、5万円の賠償命令

**教訓**: 購入価格ではなく、時価で賠償する。

### 事例2: 紛失で賠償請求（8万円の上限）

**概要**:
- クリーニング店が、コート（購入価格30万円、クリーニング代2,000円）を紛失
- 顧客が「30万円を賠償しろ」と請求
- クリーニング店が「賠償上限は8万円（クリーニング代の40倍）」と回答
- 裁判所が「賠償基準の上限を適用」として、8万円の賠償命令

**教訓**: 賠償額には上限がある。

### 事例3: 経年劣化で賠償責任なし

**概要**:
- クリーニング店が、古いワイシャツをクリーニング
- 襟が破れた
- 顧客が「賠償しろ」と請求
- クリーニング店が「経年劣化」と主張
- 裁判所が「クリーニングが原因ではない」として、請求棄却

**教訓**: 受付時に衣類の状態を確認し、記録する。
