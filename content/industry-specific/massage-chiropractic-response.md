## この記事は誰のため？

この記事は、整体院・マッサージ店・鍼灸院・リラクゼーションサロンの経営者で、Googleマップや口コミサイトで「施術効果がなかった」「痛みが悪化した」「無資格者が施術している」と書かれてしまった方に向けて書かれています。

整体・マッサージ業界の口コミは、施術効果への期待値が高く、効果が感じられないと低評価につながります。また、医師法・あはき法（あん摩マッサージ指圧師、はり師、きゅう師等に関する法律）の規制があり、表現に注意が必要です。

この記事を読むことで、施術効果クレームへの返信テンプレート、法令遵守の表現方法、予防策が分かります。

## 整体・マッサージ業界特有の口コミトラブル

### トラブル1: 施術効果への不満

**よくあるクレーム**:
- 「肩こりが治らなかった」
- 「1回で良くなると言われたのに、変わらない」
- 「高い料金を払ったのに効果なし」

**原因**:
- 施術者の過剰な効果説明
- 顧客の期待値が高すぎる
- 個人差による効果の違い

### トラブル2: 痛み・症状の悪化

**よくあるクレーム**:
- 「施術後、痛みが悪化した」
- 「揉み返しがひどい」
- 「骨をボキボキされて怪我をした」

**原因**:
- 施術者の技術不足
- 顧客の体質・既往症の見落とし
- 強すぎる施術

### トラブル3: 無資格施術の疑い

**よくあるクレーム**:
- 「資格を持っていない人が施術している」
- 「あん摩マッサージ指圧師の免許がないのにマッサージと表記」
- 「医療行為をしているのでは？」

**法的リスク**:
- あはき法違反（無資格施術）
- 医師法違反（医業類似行為）

### トラブル4: 高額な回数券・コース勧誘

**よくあるクレーム**:
- 「しつこく回数券を勧誘された」
- 「断りにくい雰囲気で契約させられた」
- 「クーリングオフができないと言われた」

**法的リスク**:
- 特定商取引法違反（不実告知、威迫行為）
- 消費者契約法違反

## 公開返信テンプレート（整体・マッサージ店用）

### テンプレート1: 施術効果への不満

```
この度はご来店いただき、誠にありがとうございました。

施術効果につきまして、ご期待に添えず申し訳ございませんでした。

当院では、お客様の症状に合わせた施術を心がけておりますが、体質や症状により効果には個人差がございます。

もしご不明な点や追加の施術をご希望の場合は、お気軽にお問い合わせください。

[店舗名] 院長
```

**ポイント**:
- 「治る」「治療」などの医療表現を使わない（医師法違反のリスク）
- 「効果には個人差がある」と明記
- 再来店を促す

### テンプレート2: 痛み・症状の悪化

```
この度はご来店いただき、誠にありがとうございました。

施術後に痛みが悪化したとのこと、大変申し訳ございませんでした。

お客様の症状につきまして、詳しくお話をお伺いしたく、恐れ入りますが店舗までご連絡いただけますでしょうか。

今後、カウンセリングをより丁寧に行い、お客様の体質に合わせた施術を徹底してまいります。

[店舗名] 院長
```

**ポイント**:
- 謝罪を明確に
- 非公開での連絡を促す（詳細な症状は公開の場で話さない）
- 改善策を明記

### テンプレート3: 無資格施術の疑い

```
この度はご来店いただき、ありがとうございました。

当院のスタッフは、全員が国家資格（あん摩マッサージ指圧師・柔道整復師等）または民間資格（整体師・セラピスト等）を保有しております。

資格情報は店内およびホームページに掲示しておりますので、ご確認いただけますと幸いです。

[店舗名] 院長
```

**ポイント**:
- 資格保有を明記
- 国家資格と民間資格を明確に区別
- 資格情報の掲示場所を案内

### テンプレート4: 高額な回数券勧誘

```
この度はご来店いただき、ありがとうございました。

回数券のご案内につきまして、ご不快な思いをさせてしまい申し訳ございませんでした。

当院では、無理な勧誘は行わないよう徹底しておりますが、スタッフの説明が不十分であった可能性がございます。

今後、お客様のご意向を最優先に、丁寧な説明を心がけてまいります。

[店舗名] 院長
```

**ポイント**:
- 無理な勧誘をしていないことを明記
- スタッフ教育の徹底を約束
- 顧客の意向を尊重する姿勢を示す

## 絶対に書いてはいけないNG返信表現

整体・マッサージ業界には、法律で禁止されている表現があります。

### NG例1: 医療表現を使う

```
NG返信: 「当院の施術で、肩こりが治ります」
```

**リスク**:
- 医師法17条違反（医業類似行為）
- あはき法違反
- 薬機法違反（治療効果の標榜）

**正しい表現**:
```
OK返信: 「当院の施術で、肩こりの症状が和らぐことがあります」
```

### NG例2: 効果を断定する

```
NG返信: 「1回で肩こりが治ります」
```

**リスク**:
- 景品表示法違反（優良誤認）
- 効果が出なかった場合、詐欺として訴えられるリスク

**正しい表現**:
```
OK返信: 「効果には個人差がございます」
```

### NG例3: 顧客の体質を批判する

```
NG返信: 「お客様の体質が原因です。当院の責任ではありません」
```

**リスク**:
- 「責任逃れ」と批判され、炎上
- 他の閲覧者にも悪印象

**正しい表現**:
```
OK返信: 「体質や症状により、効果には個人差がございます」
```

### NG例4: 医師・病院を批判する

```
NG返信: 「病院では治らないので、当院に来てください」
```

**リスク**:
- 医師法違反
- 他の医療機関への誹謗中傷

**正しい表現**:
```
OK返信: 「当院では、リラクゼーション目的の施術を提供しております」
```

## 法令遵守のための表現ガイド

整体・マッサージ業界で使える表現と使えない表現の一覧です。

### 医師法・あはき法で禁止されている表現

| 禁止表現 | 理由 | 代替表現 |
|---------|------|---------|
| 治療 | 医師法17条違反 | 施術・ケア |
| 診断 | 医師法17条違反 | カウンセリング・問診 |
| 治る | 医師法17条違反 | 和らぐ・楽になる |
| 治す | 医師法17条違反 | サポートする・整える |
| 医療マッサージ | あはき法違反（無資格の場合） | リラクゼーションマッサージ |
| 腰痛を治します | 医師法17条違反 | 腰の張りをほぐします |
| 骨盤矯正 | 医師法17条違反（医療行為と誤認） | 骨盤ケア・骨盤調整 |

### 景品表示法で禁止されている表現

| 禁止表現 | 理由 | 代替表現 |
|---------|------|---------|
| 1回で肩こりが治る | 効果の断定（優良誤認） | 効果には個人差がございます |
| 医学的に証明された | 根拠のない表現 | お客様の体験談として |
| 100%改善 | 効果の断定 | 多くのお客様にご満足いただいております |

## 整体・マッサージ店の資格と法律

### 国家資格と民間資格の違い

| 資格名 | 種類 | できること | できないこと |
|-------|------|----------|-------------|
| あん摩マッサージ指圧師 | 国家資格 | あん摩・マッサージ・指圧 | 医療行為（診断・治療） |
| 鍼灸師（はり師・きゅう師） | 国家資格 | 鍼・灸 | 医療行為（診断・治療） |
| 柔道整復師 | 国家資格 | 骨折・脱臼の応急処置、捻挫・打撲の施術 | 骨折・脱臼の本格的治療（医師の同意が必要） |
| 整体師 | 民間資格 | リラクゼーション目的の整体 | マッサージ（あん摩マッサージ指圧師の独占業務） |
| セラピスト | 民間資格 | リラクゼーション目的の施術 | マッサージ（あん摩マッサージ指圧師の独占業務） |

### 無資格施術のリスク

**あはき法12条違反**:
- あん摩マッサージ指圧師、はり師、きゅう師でない者が、業としてあん摩、マッサージ、指圧、はり、きゅうを行うことは禁止
- 違反した場合: 50万円以下の罰金

**医師法17条違反**:
- 医師でない者が、医業（診断・治療）を行うことは禁止
- 違反した場合: 3年以下の懲役もしくは100万円以下の罰金

## 予防策: 施術効果クレームを減らすために

### 予防策1: カウンセリングの徹底

施術前に、以下の内容を必ず確認します。

**確認すべき内容**:
- 症状の詳細（いつから、どの部位、どのような痛み）
- 既往症（ヘルニア、骨粗鬆症、心臓病等）
- アレルギー（オイル、アロマ等）
- 妊娠の有無
- 施術への期待値（「1回で治したい」等の過度な期待を下げる）

**カウンセリングシートの記入**:
- 上記内容を記録
- 顧客の署名をもらう（「既往症を申告した」という証拠）

### 予防策2: 効果説明の適正化

施術前に、以下の内容を必ず説明します。

**説明すべき内容**:
- 施術の内容（どの部位を、どのように施術するか）
- 期待できる効果（「和らぐ」「楽になる」等の表現）
- 効果の個人差（「全員に効果があるわけではない」）
- 揉み返しのリスク（「施術後、一時的に痛みが出ることがある」）
- 通院の目安（「症状により、複数回の施術が必要な場合がある」）

**NG説明**:
- 「1回で治ります」
- 「絶対に良くなります」
- 「病院より効果があります」

### 予防策3: 資格情報の明示

店内・ホームページ・Googleマイビジネスに、資格情報を明記します。

**明記すべき内容**:
- 施術者の氏名
- 保有資格（あん摩マッサージ指圧師、柔道整復師、整体師等）
- 資格証のコピー（店内掲示）

**Googleマイビジネスへの記載例**:
```
【施術者の資格】

・院長 〇〇: あん摩マッサージ指圧師（国家資格）
・スタッフ △△: 柔道整復師（国家資格）
・スタッフ □□: 整体師（民間資格）

当院では、国家資格保有者が責任を持って施術いたします。
```

### 予防策4: 回数券勧誘の適正化

**適正な勧誘方法**:
- 「回数券もございますが、いかがですか？」と軽く案内
- 「無理にお勧めするものではございません」と明記
- 断られたら、すぐに引き下がる

**NG勧誘**:
- 「今日契約しないと損ですよ」（威迫行為）
- 「他のお客様は皆さん買ってます」（不実告知）
- 「1回では治らないので、回数券を買ってください」（断定的判断の提供）

**特定商取引法の遵守**:
- 回数券販売は「特定継続的役務提供」に該当する場合がある
- クーリングオフ（8日間）の説明を必ず行う
- 契約書面の交付

## 整体・マッサージ業界の判例・事例

### 事例1: 施術後の痛み悪化で損害賠償請求（慰謝料50万円）

**概要**:
- 整体院で施術を受けた後、腰痛が悪化
- 顧客が損害賠償請求
- 施術者の説明不足、カウンセリング不足が認められた

**判決**: 慰謝料50万円、治療費10万円の賠償命令

**教訓**: カウンセリングを徹底し、既往症を確認する。

### 事例2: 無資格マッサージであはき法違反（罰金50万円）

**概要**:
- あん摩マッサージ指圧師の資格がないのに「マッサージ」と表記
- 保健所が指導、刑事告発
- あはき法12条違反で有罪

**判決**: 罰金50万円

**教訓**: 無資格の場合、「マッサージ」ではなく「リラクゼーション」と表記。

### 事例3: 「1回で治る」広告で景品表示法違反（措置命令）

**概要**:
- ホームページに「1回で肩こりが治る」と表記
- 消費者庁が景品表示法違反（優良誤認）として措置命令

**教訓**: 効果を断定しない。「個人差がある」と明記。
