## この記事は誰のため?

この記事は、「成績が上がらない」「効果がない」という口コミに対して、どう返信すべきか分からないとお悩みの学習塾・スクールのオーナー・塾長に向けて書かれています。

「講師の質が悪い」「授業が分かりにくい」という講師クレームへの対応方法が分からない方や、「絶対合格」「必ず成績が上がる」などの成果保証表現が使えないことは知っていても、何をどこまで書いていいか分からないという悩みを抱えている方も多いでしょう。

この記事を読むことで、学習塾・スクール特有のクレームへの返信テンプレートと、保護者との連携方法が分かります。

---

## 学習塾・スクールの口コミ対応で絶対に守るべきルール

学習塾・スクールの口コミ対応では、**成果を保証せず、個人差を明示する**ことが重要です。

### 学習塾・スクールのクレームで絶対にやってはいけないこと

1. **成果を保証する**: 「絶対合格」「必ず成績が上がります」
2. **生徒・保護者のせいにする**: 「家で勉強していないからです」
3. **講師の質を軽視する**: 「講師は問題ない」と反論する
4. **料金の透明性を欠く**: 「教材費は別途かかります」を後から説明

### 学習塾・スクールの口コミ対応の基本フロー

1. **謝罪**: まずご期待に沿えなかったことを謝罪
2. **個人差の説明**: 成績向上には個人差があることを説明
3. **サポート体制の提示**: 保護者との連携、補習授業などのサポート体制を示す
4. **改善の姿勢**: 講師の質向上、カリキュラムの見直しなど、改善の姿勢を示す

学習塾・スクールの運営については、[文部科学省](https://www.mext.go.jp/)や[全国学習塾協会](https://www.jja.or.jp/)でも相談を受け付けています。

## 公開返信テンプレ（学習塾・スクールクレーム対応）

学習塾・スクールの口コミ返信では、**成果の個人差を説明し、保護者との連携を強化する姿勢を示す**ことが重要です。

### 返信テンプレート（「成績が上がらない」への対応）

```
この度は、ご期待に沿う結果をご提供できず、申し訳ございませんでした。

当塾では、生徒様一人ひとりに合わせた指導を心がけておりますが、成績向上には個人差がございます。
また、塾での学習に加えて、ご家庭での学習習慣も重要となります。

もしご不明な点やご相談がございましたら、保護者面談を実施いたしますので、お気軽にお問い合わせください。

[塾・スクール名] 塾長
```

### 返信テンプレート（「講師の質が悪い」への対応）

```
この度は、講師の指導に関してご期待に沿えず、申し訳ございませんでした。

ご指摘の点について、講師全員で共有し、指導方法の改善に取り組んでおります。
生徒様に分かりやすい授業をご提供できるよう、講師研修を強化してまいります。

貴重なご意見をいただき、ありがとうございました。

[塾・スクール名] 塾長
```

### 返信テンプレート（「授業料が高い」への対応）

```
この度は、ご期待に沿えず申し訳ございませんでした。

当塾の授業料には、授業料、教材費、テスト対策費が含まれております。
料金体系につきましては、入塾時にご説明させていただいておりますが、ご不明な点がございましたらお気軽にお問い合わせください。

今後も、質の高い指導をご提供できるよう、努めてまいります。

[塾・スクール名] スタッフ一同
```

### 返信テンプレート（「授業が分かりにくい」への対応）

```
この度は、授業が分かりにくかったとのこと、申し訳ございませんでした。

ご指摘の点について、担当講師と共有し、よりわかりやすい授業を心がけるよう指導いたしました。
もしご不明な点がございましたら、補習授業も実施しておりますので、お気軽にお申し出ください。

生徒様の学力向上のため、スタッフ一同でサポートしてまいります。

[塾・スクール名] 塾長
```

### ポイント解説

- **「成績向上には個人差がございます」**: 成果を保証せず、個人差があることを明示
- **「保護者面談を実施いたします」**: 保護者との連携を強化する姿勢を示す
- **「講師研修を強化してまいります」**: 講師の質向上に取り組む姿勢を示す

## 絶対に使ってはいけないNG表現

以下の表現は、学習塾・スクールの口コミ対応では特に避けるべきです。

### NG例1: 成果の保証

```
❌ 「必ず成績が上がります」
❌ 「絶対に合格できます」
❌ 「〇〇点アップを保証します」
```

**リスク**: 成果を保証する表現は、景品表示法違反や誇大広告に該当する可能性がある

**正しい表現:**
```
✅ 「成績向上には、個人差がございます」
✅ 「生徒様一人ひとりに合わせた指導を心がけております」
```

### NG例2: 生徒・保護者のせいにする

```
❌ 「お子様が家で勉強していないからです」
❌ 「保護者様のサポートが不足しています」
❌ 「やる気がないのでは」
```

**リスク**: 責任転嫁は、炎上のもと。たとえ事実でも、公開返信では書かない

**正しい表現:**
```
✅ 「塾での学習に加えて、ご家庭での学習習慣も重要となります」
✅ 「保護者様と連携し、お子様の学習をサポートしてまいります」
```

### NG例3: 講師の質を軽視する

```
❌ 「講師は問題ないと認識しております」
❌ 「お子様の理解力が不足しているのでは」
```

**リスク**: 講師を擁護すると、「この塾は保護者の声を聞かない」という印象を与える

**正しい表現:**
```
✅ 「講師の指導方法について、改善に取り組んでおります」
✅ 「よりわかりやすい授業をご提供できるよう、講師研修を強化してまいります」
```

### NG例4: 料金の透明性を欠く

```
❌ 「教材費は別途かかります」（入塾時に説明していない）
❌ 「夏期講習は追加料金です」（事前に説明していない）
```

**リスク**: 料金の事後説明は、トラブルのもと

**正しい表現:**
```
✅ 「料金体系につきましては、入塾時にご説明させていただいております」
✅ 「追加費用が発生する場合は、事前にご案内しております」
```

## 学習塾・スクール特有の口コミパターンと対応

### パターン1: 「個別指導なのに、他の生徒と一緒だった」

**返信例:**
```
この度は、ご説明が不十分で申し訳ございませんでした。

当塾の個別指導は、講師1名に対して生徒様〇名までの形式となっております。
完全1対1の指導をご希望の場合は、別途マンツーマンコースもご用意しております。

今後は、入塾時により詳しくご説明させていただきます。
ご不明な点がございましたら、お気軽にお問い合わせください。
```

### パターン2: 「宿題が多すぎる」

**返信例:**
```
この度は、ご負担をおかけしてしまい申し訳ございませんでした。

当塾では、学力定着のため、一定量の宿題を出しておりますが、お子様の状況に応じて調整することも可能です。
保護者面談を実施いたしますので、お気軽にご相談ください。

お子様の学習状況に合わせて、最適な学習プランをご提案いたします。
```

### パターン3: 「自習室が使えなかった」

**返信例:**
```
この度は、自習室のご利用にご不便をおかけし、申し訳ございませんでした。

自習室は、授業日以外でもご利用いただけますが、混雑時はお席をご用意できない場合がございます。
座席の拡充を検討しており、より多くの生徒様にご利用いただけるよう努めてまいります。

お急ぎの場合は、事前にご予約いただくことも可能ですので、お問い合わせください。
```

### パターン4: 「志望校に合格できなかった」

**返信例:**
```
この度は、ご期待に沿う結果をご提供できず、誠に申し訳ございませんでした。

当塾では、生徒様一人ひとりに合わせた受験指導を行っておりますが、合格には様々な要因が影響いたします。
今後は、より効果的な受験対策を実施できるよう、カリキュラムの見直しを進めてまいります。

貴重なご意見をいただき、ありがとうございました。
```

### パターン5: 「講師がコロコロ変わる」

**返信例:**
```
この度は、講師の変更によりご不安を与えてしまい、申し訳ございませんでした。

当塾では、講師の都合により担当が変更になる場合がございますが、引き継ぎは徹底しております。
今後は、できる限り担当講師を固定できるよう、スケジュール調整に努めてまいります。

ご不安な点がございましたら、お気軽にご相談ください。
```

## 店舗としてやるべき記録

学習塾・スクールでは、指導記録・保護者面談記録を残すことで、トラブル防止につながります。

### 記録すべき項目

- **入塾時の説明記録**: 料金体系、指導形式、退塾規約の説明履歴
- **指導記録**: いつ、何を指導したか、テスト結果
- **宿題・課題の記録**: いつ、何を出したか、提出状況
- **保護者面談記録**: いつ、どんな相談があったか、どう対応したか
- **口コミ対応記録**: いつ、誰が、どう返信したか

### 入塾時の記録の取り方

```
【入塾時説明記録】
日時: 2025年11月13日
生徒名: [イニシャル]
学年: 中学2年生
希望コース: 個別指導（週2回）
説明内容:
- 個別指導は講師1名に対して生徒2〜3名と説明
- 授業料（月額〇〇円）、教材費（〇〇円）を説明
- テスト対策費は別途かかることを説明
- 退塾は1ヶ月前までに申し出が必要と説明
- 成績向上には個人差があることを説明
- 保護者との連携が重要であることを説明
同意: 入塾申込書にサイン済み
```
