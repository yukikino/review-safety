## この記事は誰のため？

この記事は、不動産会社・賃貸管理会社の経営者で、Googleマップや口コミサイトで「おとり物件を紹介された」「敷金を返してくれない」「騒音がひどい」と書かれてしまった方に向けて書かれています。

不動産業界の口コミは、トラブルが高額化しやすく、宅建業法・消費者契約法などの法律違反を疑われるケースが多くあります。適切に対応しないと、行政処分のリスクもあります。

この記事を読むことで、不動産特有のクレームへの返信テンプレート、法令遵守の方法、トラブル予防策が分かります。

## 不動産・賃貸業界特有の口コミトラブル

### トラブル1: おとり物件・虚偽広告

**よくあるクレーム**:
- 「ネットで見た物件が『既に契約済み』と言われた」
- 「広告と実際の物件が違う（写真詐欺）」
- 「『礼金なし』と書いてあったのに、実際は礼金1ヶ月」

**法的リスク**:
- 宅建業法47条違反（誇大広告の禁止）
- 景品表示法違反（有利誤認・優良誤認）
- 消費者契約法違反（不実告知）

### トラブル2: 設備不良・修繕対応の遅れ

**よくあるクレーム**:
- 「エアコンが壊れているのに、1ヶ月放置された」
- 「水漏れを報告したのに対応してくれない」
- 「入居前の清掃が不十分（ゴミが残っている）」

**法的リスク**:
- 賃貸借契約の債務不履行
- 民法606条（修繕義務）違反

### トラブル3: 騒音・近隣トラブル

**よくあるクレーム**:
- 「上の階の足音がうるさい」
- 「隣の部屋の住民がうるさいのに、管理会社が何もしてくれない」
- 「ゴミ出しルールを守らない住民がいる」

**管理責任**:
- 管理会社には、近隣トラブルの仲裁義務がある
- 放置すると、契約解除や損害賠償請求のリスク

### トラブル4: 敷金・原状回復費用の高額請求

**よくあるクレーム**:
- 「敷金が全額返ってこない」
- 「クリーニング代として10万円請求された」
- 「壁紙の張替え費用として20万円請求された」

**法的リスク**:
- 国土交通省「原状回復ガイドライン」違反
- 消費者契約法10条違反（消費者の利益を一方的に害する条項）

### トラブル5: 仲介手数料・初期費用の高額請求

**よくあるクレーム**:
- 「仲介手数料が家賃の1.1ヶ月分（消費税込）を超えている」
- 「鍵交換費用が5万円と高額」
- 「24時間サポート費用が必須と言われた」

**法的リスク**:
- 宅建業法46条違反（報酬の制限）
- 消費者契約法違反（不当な条項）

## 公開返信テンプレート（不動産・賃貸用）

### テンプレート1: おとり物件疑惑

```
この度はお問い合わせいただき、誠にありがとうございました。

ご覧いただいた物件につきまして、お問い合わせ時点で既に申込が入っており、ご案内できず申し訳ございませんでした。

当社では、最新の空室情報を随時更新しておりますが、申込のタイミングにより、ご案内できない場合がございます。

今後、より迅速な情報更新を徹底してまいります。

[会社名] 担当: 〇〇
```

**ポイント**:
- 「既に申込が入っていた」という事実を明記
- 「おとり物件」という言葉は使わない（認めたことになる）
- 情報更新の遅れを認め、改善を約束

### テンプレート2: 設備不良・修繕対応の遅れ

```
この度は設備の不具合につきまして、対応が遅れ大変申し訳ございませんでした。

ご指摘いただいたエアコンの修理は、〇月〇日に完了しております。

今後、修繕依頼を受けた際は、迅速に対応するよう社内体制を改善いたしました。

ご不便をおかけしましたこと、重ねてお詫び申し上げます。

[会社名] 担当: 〇〇
```

**ポイント**:
- 謝罪を明確に
- 対応完了の事実を明記
- 改善策を具体的に示す

### テンプレート3: 騒音・近隣トラブル

```
この度は騒音につきまして、ご不快な思いをさせてしまい申し訳ございません。

該当の住戸に対しまして、管理規約に基づき注意喚起の文書を投函いたしました。

引き続き、快適にお住まいいただけるよう、巡回・注意喚起を徹底してまいります。

[会社名] 担当: 〇〇
```

**ポイント**:
- 対応したことを明記（「何もしていない」と思われないように）
- 具体的な対応内容を記載
- プライバシーに配慮（「〇号室の住民」など特定しない）

### テンプレート4: 敷金返還トラブル

```
この度は敷金の返還につきまして、ご不明な点があり申し訳ございません。

原状回復費用の内訳につきまして、国土交通省のガイドラインに基づき算定しております。

詳細をご説明させていただきたく、お手数ですが店舗までご連絡いただけますでしょうか。

[会社名] 担当: 〇〇
```

**ポイント**:
- 「国土交通省ガイドライン」を明記（法的根拠を示す）
- 公開の場で金額を議論しない
- 非公開での連絡を促す

### テンプレート5: 仲介手数料の高額請求

```
この度はお問い合わせいただき、ありがとうございました。

仲介手数料につきまして、宅建業法に基づき、家賃の1.1ヶ月分（消費税込）以内で設定しております。

内訳の詳細につきまして、ご説明させていただきたく、お手数ですが店舗までご連絡いただけますでしょうか。

[会社名] 担当: 〇〇
```

**ポイント**:
- 法的根拠（宅建業法）を明記
- 金額が適正であることを示す
- 詳細は非公開で説明

## 絶対に書いてはいけないNG返信表現

不動産業界の返信には、炎上リスクの高いNG表現があります。

### NG例1: おとり物件を認める

```
NG返信: 「その物件は、客寄せ用の物件です」
```

**リスク**:
- 宅建業法違反を自白
- 行政処分（業務停止・免許取消）のリスク

**正しい対応**:
```
OK返信: 「お問い合わせ時点で既に申込が入っており、ご案内できませんでした」
```

### NG例2: 入居者のプライバシーを晒す

```
NG返信: 「〇号室の住民が騒音の原因です」
```

**リスク**:
- プライバシー侵害
- 逆に〇号室の住民から訴えられる

**正しい対応**:
```
OK返信: 「該当の住戸に対し、注意喚起いたしました」
```

### NG例3: 敷金返還を拒否する

```
NG返信: 「契約書に『敷金は返さない』と書いてあります」
```

**リスク**:
- 消費者契約法10条違反（消費者の利益を一方的に害する条項は無効）
- 「悪徳業者」として炎上

**正しい対応**:
```
OK返信: 「原状回復ガイドラインに基づき、適正に算定しております」
```

### NG例4: 顧客を批判する

```
NG返信: 「そもそも、お客様が部屋を汚したのが悪いです」
```

**リスク**:
- 顧客批判は炎上のもと
- 他の閲覧者にも悪印象

**正しい対応**:
```
OK返信: 「詳細をご説明させていただきたく、店舗までご連絡ください」
```

## 不動産業界の法令遵守ガイド

### 宅建業法で禁止されている行為

| 禁止行為 | 条文 | 罰則 |
|---------|------|------|
| 誇大広告 | 47条 | 6ヶ月以下の懲役または100万円以下の罰金 |
| おとり物件 | 47条 | 同上 |
| 重要事項の不告知 | 35条 | 業務停止処分 |
| 報酬の上限超過 | 46条 | 業務停止処分 |

### 国土交通省「原状回復ガイドライン」

**基本原則**:
- 通常の使用による損耗（経年劣化）: 貸主負担
- 借主の故意・過失による損耗: 借主負担

**貸主負担の例**:
- 畳の日焼け
- 壁紙の自然な汚れ・黄ばみ
- フローリングの色落ち（日焼け）
- 冷蔵庫の後ろの壁の黒ずみ（電気ヤケ）

**借主負担の例**:
- タバコのヤニ・臭い
- ペットによる傷・臭い
- 故意に壊した設備
- 清掃を怠ったことによるカビ・シミ

**計算方法**:
- 減価償却を考慮
- 例: 壁紙の耐用年数は6年。3年住んだ場合、残存価値は50%。全額請求は違法。

## 予防策: 不動産トラブルを減らすために

### 予防策1: 物件情報の即時更新

おとり物件を疑われないために、空室情報を即座に更新します。

**更新すべきタイミング**:
- 申込が入った時点で「申込中」に変更
- 契約完了後、速やかに削除

**推奨ツール**:
- 不動産ポータルサイトの自動更新機能
- 社内の空室管理システム

### 予防策2: 重要事項の事前説明

契約前に、以下の内容を必ず説明します。

**説明すべき内容**:
- 物件の設備状況（エアコン、給湯器等の築年数）
- 騒音リスク（上下階・隣室の状況）
- 修繕履歴（過去の水漏れ、設備故障等）
- 周辺環境（ゴミ置き場、駐輪場等）
- 退去時の原状回復費用の目安

**重要事項説明書への記載**:
- 上記内容を書面に明記
- 顧客の署名・捺印をもらう

### 予防策3: 修繕対応のスピードアップ

**修繕依頼を受けた場合の対応フロー**:
1. 依頼受付後、24時間以内に現地確認
2. 修繕業者の手配（緊急性に応じて即日〜3日以内）
3. 修繕完了の報告

**緊急修繕の基準**:
- 水漏れ: 即日対応
- エアコン故障（真夏・真冬）: 即日〜翌日対応
- その他設備故障: 3日以内

### 予防策4: 原状回復費用の透明化

退去時のトラブルを防ぐため、原状回復費用を事前に説明します。

**入居時の説明**:
- 「国土交通省ガイドライン」に基づくこと
- 通常損耗は請求しないこと
- 借主負担となるケース（タバコ、ペット等）

**退去立会いの実施**:
- 借主立会いのもと、部屋の状況を確認
- 写真撮影（証拠として保管）
- 原状回復費用の見積りを提示

**精算書の送付**:
- 退去後1ヶ月以内に精算書を送付
- 敷金返還額を明記
- 原状回復費用の内訳を詳細に記載

### 予防策5: 仲介手数料の適正化

**宅建業法46条の遵守**:
- 仲介手数料は、家賃の1.1ヶ月分（消費税込）が上限
- 依頼者の承諾なく、上限を超えることは違法

**その他初期費用の透明化**:
- 鍵交換費用: 実費相当（1.5万円〜3万円程度）
- 24時間サポート費用: 任意加入であることを明記
- 火災保険: 複数プランを提示し、選択肢を与える

## 不動産業界の判例・事例

### 事例1: おとり物件で業務停止処分

**概要**:
- 不動産会社が、既に契約済みの物件をネットで掲載し続けた
- 国土交通省が宅建業法違反（誇大広告）として業務停止処分

**処分**: 業務停止3ヶ月

**教訓**: 空室情報は即座に更新する。

### 事例2: 敷金返還トラブルで訴訟（全額返還命令）

**概要**:
- 不動産会社が、通常損耗（壁紙の黄ばみ等）を理由に敷金を全額差し引いた
- 借主が提訴
- 裁判所が「通常損耗は貸主負担」として、敷金の全額返還命令

**判決**: 敷金20万円の全額返還 + 遅延損害金

**教訓**: 原状回復ガイドラインを遵守する。

### 事例3: 騒音クレーム放置で損害賠償請求（50万円）

**概要**:
- 管理会社が、騒音クレームを半年間放置
- 借主が「安眠妨害」として損害賠償請求
- 裁判所が「管理義務違反」として損害賠償命令

**判決**: 慰謝料50万円

**教訓**: 騒音クレームは放置せず、速やかに対応する。
