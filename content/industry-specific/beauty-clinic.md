# 美容クリニック向け口コミ対応｜薬機法に違反しない返信文の書き方

## この記事は誰のため?

この記事は、以下のような状況でお困りの美容クリニックのオーナー・院長に向けて書かれています：

- 「効果がなかった」「高額だった」という口コミに返信したいが、薬機法に違反しないか不安
- 「必ず痩せます」「シワが消えます」といった表現が使えないことは知っているが、何をどこまで書いていいか分からない
- 他の美容クリニックの返信例を見ても、どれが正しいのか判断できない

この記事を読むことで、**薬機法・医療法に違反しない安全な返信テンプレートと、絶対に使ってはいけないNG表現**が分かります。

---

## 美容クリニックの口コミ対応で絶対に守るべきルール

美容クリニックは医療機関であり、口コミへの返信も**医療広告ガイドライン**の対象になります。

### 医療広告ガイドラインの主な禁止事項

1. **効果の断定**: 「必ず痩せます」「シワが消えます」
2. **ビフォーアフター写真**: 口コミ返信に写真を添付する行為（患者本人の承諾があっても原則NG）
3. **比較優良広告**: 「他院より安い」「業界No.1」
4. **著名人の推薦**: 「芸能人も通っています」
5. **体験談の誇張**: 「全員が満足しています」

### 口コミ返信でやってはいけないこと

- **患者の個人情報を書く**: 「〇〇様は〜」と実名を出す
- **効果を保証する**: 「次回は必ず〜」と断定する
- **他の患者の成功例を引用**: 「他の患者様は〜」と比較する

## 公開返信テンプレ（薬機法対応）

美容クリニックの口コミ返信では、**事実のみを簡潔に伝え、効果の断定は避ける**ことが重要です。

### 返信テンプレート（「効果がなかった」への対応）

```
この度は、ご期待に沿う結果をご提供できず、申し訳ございませんでした。

当院の治療は、患者様の体質や生活習慣により、効果の実感に個人差がございます。
術前のカウンセリングでご説明が不足していた点がございましたら、重ねてお詫び申し上げます。

もしご不明な点やご相談がございましたら、直接当院までお問い合わせいただけますと幸いです。

[クリニック名] 院長
```

### 返信テンプレート（「高額だった」への対応）

```
この度は、ご期待に沿えず申し訳ございませんでした。

当院の治療費には、診察料、施術料、術後のフォローアップが含まれております。
料金体系につきましては、カウンセリング時にご説明させていただいておりますが、ご不明な点がございましたらお気軽にお問い合わせください。

今後も患者様にご納得いただける治療を提供できるよう、努めてまいります。

[クリニック名] スタッフ一同
```

### 返信テンプレート（「痛かった」への対応）

```
この度は、施術中に痛みを感じさせてしまい、申し訳ございませんでした。

当院では、患者様の痛みに配慮した施術を心がけておりますが、痛みの感じ方には個人差がございます。
今後は、施術前により詳しく痛みの程度をご説明し、ご不安を軽減できるよう努めてまいります。

ご意見をいただき、ありがとうございました。

[クリニック名] 院長
```

### ポイント解説

- **「効果の実感には個人差がございます」**: 効果を断定せず、個人差があることを明示
- **「ご期待に沿えず」**: 「効果がなかった」という表現をそのまま認めるのではなく、「期待に沿えなかった」と柔らかく受け止める
- **料金の透明性**: 何に対して料金がかかっているかを簡潔に説明（長すぎる説明は逆効果）

## 絶対に使ってはいけないNG表現（薬機法違反）

以下の表現は、薬機法・医療法に違反する可能性があるため、絶対に使わないでください。

### NG例1: 効果の断定

```
❌ 「次回ご来院いただければ、必ず痩せられます」
❌ 「当院の治療で、シワが消えます」
❌ 「100%満足いただけます」
```

**リスク**: 効果を断定する表現は、医療広告ガイドライン違反。行政指導・罰則の対象になる

**正しい表現:**
```
✅ 「次回ご来院時に、改めてカウンセリングさせていただきます」
✅ 「当院では、患者様一人ひとりに合わせた治療をご提案しております」
```

### NG例2: ビフォーアフターの提示

```
❌ 「他の患者様のビフォーアフター写真をお見せします」
❌ 「こちらの写真をご覧ください」（写真を添付）
```

**リスク**: ビフォーアフター写真は、医療広告ガイドラインで原則禁止（限定的に認められる場合を除く）

**正しい表現:**
```
✅ 「カウンセリング時に、治療内容を詳しくご説明させていただきます」
```

### NG例3: 比較優良広告

```
❌ 「他院より安い料金で提供しております」
❌ 「業界No.1の実績」
❌ 「〇〇地域で唯一の〜」
```

**リスク**: 比較優良広告は、医療広告ガイドライン違反

**正しい表現:**
```
✅ 「当院の料金体系は、ホームページでご確認いただけます」
```

### NG例4: 著名人の推薦

```
❌ 「芸能人の〇〇さんも当院に通っています」
❌ 「有名人も満足されています」
```

**リスク**: 著名人の推薦は、医療広告ガイドライン違反

### NG例5: 患者の個人情報

```
❌ 「〇〇様は、3回目の施術で効果を実感されました」
```

**リスク**: 患者の個人情報（実名）を公開することは、プライバシー侵害

**正しい表現:**
```
✅ 「効果の実感には、施術回数や個人差がございます」
```

## 美容クリニック特有の口コミパターンと対応

### パターン1: 「カウンセリングで高額プランを勧められた」

**返信例:**
```
当院では、患者様のご希望をお伺いした上で、複数のプランをご提案しております。
もしご説明が不十分だった点がございましたら、お詫び申し上げます。

ご予算やご希望に合わせたプランもご用意しておりますので、お気軽にご相談ください。
```

### パターン2: 「ダウンタイムの説明が不足していた」

**返信例:**
```
ダウンタイムにつきまして、ご説明が不足しており申し訳ございませんでした。

当院では、術前のカウンセリングでダウンタイムの期間や注意点をご説明しておりますが、今後はより詳しくお伝えできるよう改善してまいります。
```

### パターン3: 「予約が取りにくい」

**返信例:**
```
ご予約が取りにくく、ご不便をおかけして申し訳ございません。

現在、予約枠の拡大を検討しており、より多くの患者様にご利用いただけるよう努めてまいります。
```

## 店舗としてやるべき記録

美容クリニックでは、医療記録と併せて、口コミ対応の記録も残すことが重要です。

### 記録すべき項目

- **カウンセリング記録**: いつ、誰が、どんな説明をしたか
- **同意書**: 施術内容、リスク、ダウンタイムの説明に同意したか
- **施術記録**: 何回施術したか、どんな内容だったか
- **口コミ対応記録**: いつ、誰が、どう返信したか

### カウンセリング時の記録の取り方

```
【カウンセリング記録】
日時: 2025年10月15日
患者名: [イニシャル]
希望する治療: ボトックス注射
説明内容:
- 効果の実感には個人差があることを説明
- ダウンタイム（3〜7日程度）を説明
- 料金（〇〇円）を説明
- リスク（内出血、腫れ）を説明
同意: 同意書にサイン済み
```
