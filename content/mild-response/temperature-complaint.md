## この記事は誰のため？

この記事は、Googleマップや食べログで「店内が暑すぎて汗だくになった」「エアコンが効きすぎて寒い」「換気で風が強くて寒い」と書かれてしまった飲食店・カフェの経営者に向けて書かれています。

店内温度は、個人差が大きく、「暑い」と感じる人もいれば「寒い」と感じる人もいます。特にコロナ禍以降は換気が重視され、冬場に窓を開けることで「寒い」というクレームが増えています。

この記事を読むことで、温度クレームへの返信テンプレート、空調調整の判断基準、換気と快適性の両立方法が分かります。

## 公開返信テンプレ（温度クレーム用）

![店内温度管理システム](./images/temperature-management-system.png)

温度クレームの返信では、温度調整と代替案を提示します。

### 返信テンプレート（店内が暑い）

```
この度はご来店いただき、ありがとうございました。

店内の温度につきまして、ご不快な思いをさせてしまい申し訳ございませんでした。

ご指摘を受け、空調設定を見直し、より快適な温度を保つよう調整いたしました。

またのご来店を心よりお待ちしております。

[店舗名] 店長
```

### 返信テンプレート（エアコンが寒すぎる）

```
この度はご来店いただき、誠にありがとうございました。

エアコンが効きすぎて寒かったとのこと、申し訳ございませんでした。

ご指摘を受け、空調温度を調整いたしました。また、ブランケットもご用意しておりますので、寒いと感じられた際はスタッフまでお申し付けください。

ご不便をおかけし、申し訳ございませんでした。

[店舗名] 店長
```

### 返信テンプレート（換気で寒い・風が強い）

```
この度はご来店いただき、ありがとうございました。

換気による風が強く、寒い思いをさせてしまい申し訳ございませんでした。

当店では感染症対策として換気を徹底しておりますが、ブランケットのご用意、席の移動など柔軟に対応させていただきます。

お気軽にスタッフまでお申し付けください。

[店舗名] 店長
```

### 返信テンプレート（夏場・冷房なし）

```
この度はご来店いただき、ありがとうございました。

店内の暑さにつきまして、ご不便をおかけし申し訳ございませんでした。

当店は古い建物のため、冷房設備が限定的で、ご期待に添えず申し訳ございません。扇風機の増設、冷たいおしぼりのご提供など、できる限りの対応をさせていただきます。

ご理解のほど、よろしくお願いいたします。

[店舗名] 店長
```

## 温度クレームの種類別対応

クレームの種類によって、返信内容と改善策が異なります。

### パターン1: 店内が暑い（夏場）

![店内温度管理システム](./images/temperature-management-system.png)

店内が暑い原因として、エアコンの設定温度が高い、エアコンの能力不足、厨房の熱が客席に流れている、窓際・日当たりの良い席が暑いといったことが考えられます。

このような場合、温度調整を約束し、扇風機・サーキュレーターの活用を伝えます。また、窓際席への配慮を明記することで、顧客満足度を高めます。

**返信テンプレート**:
```
この度はご来店いただき、ありがとうございました。

店内の暑さにつきまして、ご不快な思いをさせてしまい申し訳ございませんでした。

ご指摘を受け、以下の改善を行いました。

【改善内容】
- エアコンの設定温度を下げました（28℃→26℃）
- サーキュレーターを増設し、空気の循環を改善
- 窓際席には日よけカーテンを設置

快適にお過ごしいただけるよう努めてまいります。

[店舗名] 店長
```

### パターン2: エアコンが寒すぎる（夏場）

エアコンの設定温度が低すぎる、エアコンの風が直接客席に当たる、女性客・高齢者は寒さに敏感といった原因があります。

このような場合、温度調整を約束し、ブランケット提供を明記します。また、席移動の柔軟な対応を伝えることで、顧客の不安を解消できます。

**返信テンプレート**:
```
この度はご来店いただき、誠にありがとうございました。

エアコンが効きすぎて寒かったとのこと、申し訳ございませんでした。

当店では、以下の対応をさせていただいております。

【対応内容】
- ブランケットの無料貸出（お気軽にお申し付けください）
- エアコンの風が直接当たらない席へのご案内
- 温度調整（お客様のご希望に応じて）

ご遠慮なくスタッフまでお申し付けください。

[店舗名] 店長
```

### パターン3: 冬場に暖房が効かない・寒い

![店内温度管理システム](./images/temperature-management-system.png)

暖房の設定温度が低い、暖房の能力不足、換気で暖かい空気が逃げている、入口の開閉で冷気が入るといった原因があります。

このような場合、暖房温度の調整を約束し、換気との両立を説明します。また、ブランケット・温かい飲み物の提供を明記することで、誠意を示せます。

**返信テンプレート**:
```
この度はご来店いただき、ありがとうございました。

店内が寒く、ご不便をおかけし申し訳ございませんでした。

当店では感染症対策として換気を徹底しており、外気が入ることがございます。ブランケットのご用意、温かいお飲み物のサービスなど、できる限りの対応をさせていただきます。

お気軽にスタッフまでお申し付けください。

[店舗名] 店長
```

### パターン4: 換気で風が強い・寒い

換気のために窓を開けている、換気扇の風が強い、入口のドアを開けっ放しにしているといった原因があります。

このような場合、換気の必要性を説明し、風が当たらない席への移動を提案します。また、ブランケット提供を明記することで、顧客の理解を得られます。

**返信テンプレート**:
```
この度はご来店いただき、ありがとうございました。

換気による風が強く、寒い思いをさせてしまい申し訳ございませんでした。

当店では感染症対策として、常時換気を行っております。風が直接当たらない席へのご案内、ブランケットのご提供など、柔軟に対応させていただきます。

次回ご来店の際は、お気軽にスタッフまでお申し付けください。

[店舗名] 店長
```

## 絶対に書いてはいけないNG返信表現

温度クレームの返信には、炎上リスクの高いNG表現があります。

### NG例1: 顧客の体質のせいにする

「他のお客様は快適と仰っています。おそらく個人差です」という表現は、「あなただけが変」と言っているように聞こえ、レビュアーを刺激します。

### NG例2: 換気を強調しすぎる

「コロナ対策で換気が最優先です。寒いのは我慢してください」という表現は、「客の快適性を軽視している」と受け取られます。

### NG例3: 設備不足を開き直る

「当店は古い建物なので、エアコンがありません」という仕方ないと突き放す表現は、不誠実と受け取られます。

### NG例4: 厚着・薄着を指示する

「寒い場合は、厚着でお越しください」という表現は、「店側が対策すべき」と批判されます。

## 温度環境を改善する方法

![店内温度管理システム](./images/temperature-management-system.png)

クレームを減らすための具体的な改善策です。適切な温度設定として、季節・時間帯に応じて適切な温度を設定します。夏場（冷房）は26℃〜28℃で湿度50〜60%、冬場（暖房）は20℃〜22℃で湿度40〜60%、春・秋は22℃〜24℃で湿度50〜60%が推奨されます。注意点として、設定温度は目安であり、実際の体感温度は席の位置や混雑状況で変わります。

ゾーニング（温度別エリア）として、店内を温度別のエリアに分けます。エリアAはエアコンの風が強いエリア（涼しい・寒い）、エリアBはエアコンの風が弱いエリア（暖かい・普通）、エリアCは窓際エリア（夏は暑い、冬は寒い）とします。運用方法として、案内時に「暑がりですか？寒がりですか？」と確認し、顧客の好みに応じて席を振り分けます。

ブランケット・扇子の提供として、寒い・暑いと感じた顧客に、ブランケットや扇子を提供します。提供方法として、レジカウンターに常備し、各テーブルに「ブランケットあります」と掲示し、スタッフから積極的に声かけします。コストはブランケット1枚1,000円〜、扇子1本100円〜です。

サーキュレーター・扇風機の活用として、サーキュレーターで空気を循環させ、温度ムラをなくします。設置場所は、天井付近（暖かい空気を下に送る）、厨房と客席の間（熱気を逃がす）、エアコンの対角線上（空気を循環）です。効果として、エアコンの設定温度を変えずに体感温度を改善できます。

## 換気と快適性の両立

![店内温度管理システム](./images/temperature-management-system.png)

コロナ禍以降、換気が重視される一方、「寒い」というクレームも増えています。機械換気の導入として、窓を開けずに換気扇・換気システムで換気します。メリットは、窓を開けないので外気の影響が少なく、温度変化が少ないことです。デメリットは、導入コストが高い（数十万円〜）ことです。

部分的な窓開けとして、全ての窓を開けるのではなく、一部だけ開けます。開け方として、窓を全開にせず5〜10cm程度開け、入口のドアと奥の窓を開けて空気の流れを作り、客席から遠い窓を優先して開けます。効果として、換気を確保しつつ温度変化を最小限にできます。

時間帯別の換気として、営業時間中常に換気するのではなく、時間帯別に換気します。換気タイミングは、営業前（開店1時間前）は窓全開で換気し、営業中は窓を少し開けて常時換気し、ピーク時は換気扇のみ稼働で窓は閉め、営業後は窓全開で換気します。

## 温度情報の事前案内

クレームを防ぐため、温度情報を事前に明示します。店頭掲示として、店頭に換気状況や温度対策を掲示します。掲示例は、「当店では、感染症対策として常時換気を行っております。寒いと感じられた際は、ブランケットをご用意しておりますので、お気軽にスタッフまでお申し付けください。ご理解とご協力をお願いいたします」です。

ホームページ・SNSでの案内として、ホームページやSNSで空調・換気の状況を案内します。案内例は、「当店では、感染症対策として常時換気を行っております。店内が寒いと感じられる場合は、ブランケットのご用意、暖かいお席へのご案内など対応させていただきます。お気軽にスタッフまでお申し付けください」です。
