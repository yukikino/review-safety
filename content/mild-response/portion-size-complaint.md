## この記事は誰のため？

この記事は、Googleマップや食べログで「量が少なすぎてお腹いっぱいにならない」「値段の割に量が少ない」「コスパ悪い」と書かれてしまった飲食店の経営者に向けて書かれています。

料理の量は、顧客によって期待値が大きく異なります。大食いの男性客には「少ない」と感じられても、小食の女性客には「ちょうど良い」と感じられます。また、高級店と大衆店では、量よりも質を重視するか、逆に量を重視するかが異なります。

この記事を読むことで、量クレームへの返信テンプレート、大盛り対応の方法、品質と量のバランスの伝え方が分かります。

## 公開返信テンプレ（量クレーム用）

![料理提供量の改善例](./images/portion-size-improvement.png)

量クレームの返信では、量の調整可能性や品質とのバランスを説明します。

### 返信テンプレート（量が少ない・基本パターン）

```
この度はご来店いただき、ありがとうございました。

料理の量につきまして、ご期待に添えず申し訳ございませんでした。

当店では、ご飯の大盛りを無料でご提供しております。また、量の調整も可能ですので、次回ご来店の際はお気軽にスタッフまでお申し付けください。

またのご来店をお待ちしております。

[店舗名] 店長
```

### 返信テンプレート（コスパ悪い・値段に見合わない）

```
この度はご来店いただき、ありがとうございました。

価格に対するご期待に添えず、申し訳ございませんでした。

当店では、〇〇産の食材を使用し、手作りにこだわった料理をご提供しております。量よりも質を重視しておりますが、お客様のご意見を参考に、価格設定の見直しも検討してまいります。

貴重なご意見をありがとうございました。

[店舗名] 店長
```

### 返信テンプレート（ご飯・パンが少ない）

```
この度はご来店いただき、ありがとうございました。

ご飯の量につきまして、ご不満を感じさせてしまい申し訳ございませんでした。

当店では、ご飯の大盛りを無料でご提供しております。また、おかわりも可能ですので、次回はお気軽にスタッフまでお申し付けください。

ご案内が不十分で申し訳ございませんでした。

[店舗名] 店長
```

### 返信テンプレート（高級店・質重視）

```
この度はご来店いただき、ありがとうございました。

料理の量につきまして、ご意見をいただき感謝申し上げます。

当店では、〇〇の食材を使用し、一皿一皿を丁寧に調理しております。量よりも質を重視したコース構成となっておりますが、お腹いっぱいお召し上がりいただきたい場合は、追加料理もご用意しております。

次回ご来店の際は、お気軽にスタッフまでご相談ください。

[店舗名] 店長
```

## 量クレームの種類別対応

> **💡 ヒント**: 他の軽度クレームへの返信テンプレートは[軽度クレーム対応一覧](/mild-response/)で確認できます。「待たされた」「態度が悪い」「清潔感がない」など19種類のテンプレートを用意しています。料理の味クレームは[料理がまずいクレーム対応](/mild-response/bad-taste-complaint)、価格クレームは[高い割に効果がないクレーム対応](/mild-response/high-price-complaint)もご参照ください。

クレームの種類によって、返信内容と改善策が異なります。

### パターン1: 大盛り無料なのに案内不足

![料理提供量の改善例](./images/portion-size-improvement.png)

大盛り無料サービスがあるが顧客が知らない、メニューに記載していない、スタッフが案内していないという状況があります。

このような場合、大盛り無料を明記し、案内不足を認めて改善を約束します。また、メニュー記載の強化を伝えることで、今後のトラブルを防げます。

**返信テンプレート**:
```
この度はご来店いただき、ありがとうございました。

料理の量につきまして、ご案内が不十分で申し訳ございませんでした。

当店では、ご飯の大盛りを無料でご提供しております。メニューに大盛り無料の表記を追加し、スタッフからもご案内するよう改善いたしました。

次回はぜひ大盛りでお召し上がりください。

[店舗名] 店長
```

### パターン2: 男性客向けのメニューがない

女性客・ダイエット客向けの少量メニューが中心で、男性客・大食い客には量が足りない、ボリュームメニューの選択肢がないという状況があります。

このような場合、ターゲット層を説明し、追加料理・大盛りの提案をします。また、ボリュームメニューの導入を検討することで、幅広い顧客層に対応できます。

**返信テンプレート**:
```
この度はご来店いただき、ありがとうございました。

料理の量につきまして、ご満足いただけず申し訳ございませんでした。

当店は女性のお客様向けのメニュー構成となっておりますが、ご要望に応じて大盛り対応、追加料理のご提案も可能です。

また、ボリュームメニューの導入も検討してまいります。

[店舗名] 店長
```

### パターン3: 高級店・質重視で量が少ない

![料理提供量の改善例](./images/portion-size-improvement.png)

高級食材を使用し量よりも質を重視している、コース料理で一皿の量は少なめ、顧客が「高いのに量が少ない」と不満に感じている状況があります。

このような場合、品質へのこだわりを説明し、コース構成の意図を伝えます。また、追加料理の提案をすることで、顧客満足度を高めます。

**返信テンプレート**:
```
この度はご来店いただき、ありがとうございました。

料理の量につきまして、ご意見をいただき感謝申し上げます。

当店では、〇〇産の希少食材を使用し、シェフが一皿一皿を丁寧に調理しております。全8皿のコース構成で、最後まで美味しくお召し上がりいただけるよう、量を調整しております。

もしお腹いっぱいお召し上がりたい場合は、追加料理もご用意しておりますので、お気軽にスタッフまでご相談ください。

[店舗名] 料理長
```

### パターン4: コスパ重視の客層とのミスマッチ

店舗は質重視だが顧客はコスパ重視、価格帯と顧客の期待値がずれている、「高いのに量が少ない」というクレームが多いという状況があります。

このような場合、店舗のコンセプトを説明し、価格に見合う価値を伝えます。また、顧客の期待値とのミスマッチを認めることで、誠実さを示せます。

**返信テンプレート**:
```
この度はご来店いただき、ありがとうございました。

価格に対するご期待に添えず、申し訳ございませんでした。

当店では、有機野菜、無添加調味料、〇〇産の肉を使用し、手作りにこだわった料理をご提供しております。量よりも質を重視したメニュー構成となっており、コストパフォーマンスよりも、食材の安全性・美味しさを大切にしております。

お客様のご期待に添えなかったこと、お詫び申し上げます。

[店舗名] 店長
```

## 絶対に書いてはいけないNG返信表現

量クレームの返信には、炎上リスクの高いNG表現があります。

### NG例1: 顧客の食欲を否定する

「普通の人なら十分な量です。食べ過ぎではないでしょうか」という表現は、顧客の体質・食欲を否定すると必ず炎上します。

### NG例2: 他店と比較する

「ファミレスのような大盛りを期待されているなら、他店をご利用ください」という上から目線の表現は、批判されます。

### NG例3: 長文で食材へのこだわりを説明

「当店のシェフは〇〇で修行し、〇〇の食材を〇〇から取り寄せ、〇〇の調理法で…」と500文字以上の説明をすると、「反省していない」と受け取られます。

### NG例4: 追加料金を強調する

「量を増やしたい場合は、追加料金500円で対応します」という表現は、「追加料金を取るのか」と批判されます。無料対応を優先すべきです。

## 量クレームを減らす予防策

![料理提供量の改善例](./images/portion-size-improvement.png)

量クレームを未然に防ぐための対策です。メニューに量を明記として、メニューに料理の量を具体的に記載します。記載例は、「Aランチ：ご飯普通盛り（200g）、大盛り無料（300g）、メイン料理150g、サラダ100g、Bランチ：パスタ200g（大盛り+100円で300g）」です。効果として、顧客が量をイメージしやすくなり、注文時の期待値のズレを防げます。

大盛り無料を積極的にアピールとして、大盛り無料サービスをメニュー・店頭・SNSで積極的にアピールします。アピール場所は、メニューの目立つ位置に「大盛り無料」と記載、テーブルに「大盛り無料」のPOP設置、Googleマップの説明欄に記載、SNSで定期的に投稿します。

注文時にスタッフが量を確認として、注文時にスタッフが量について確認します。確認例は、「ご飯の量は、普通盛りと大盛り（無料）がございますが、いかがなさいますか？」「かしこまりました。大盛りは普通盛りの1.5倍の量になります」です。効果として、顧客が量を把握し、「少ない」というクレームを防げます。

写真で量をイメージさせるとして、メニュー写真に実際の量が分かるものを写し込みます。写真の工夫として、お茶碗を写し込む（大きさの基準）、スプーン・フォークを添える、手を写し込む（実寸がイメージしやすい）とします。NGな写真は、盛り付けを綺麗に見せるため実際より多く盛る、小さい器に盛って量が多く見えるようにすることです。

## 大盛り・追加料理の提供方法

![料理提供量の改善例](./images/portion-size-improvement.png)

量に不満を感じた顧客への対応方法です。大盛り無料の運用として、大盛り無料を提供する場合の運用ルールを明確にします。大盛り無料の範囲は、ご飯は1.5倍まで無料、麺は1.5倍まで無料、サラダは無料対応なし（原価が高いため）です。制限は、1人1回まで、テイクアウトは対象外とします。

追加料理の提案として、量が足りない顧客には追加料理を提案します。追加料理の例は、ご飯おかわり無料、唐揚げ追加+200円、サラダ追加+300円です。提案タイミングは、食事中盤にスタッフが声かけし、「お腹いっぱいですか？追加料理もご用意できます」と伝えます。
