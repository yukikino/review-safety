## この記事は誰のため？

この記事は、Instagram上で低評価コメント、炎上DM、ストーリーズでのネガティブメンションを受けた店舗・企業アカウント運営者に向けて書かれています。

Instagramは、GoogleマップやAmazonのような「レビュー機能」はありませんが、コメント欄、DM、ストーリーズメンション、ハッシュタグ投稿などで、実質的に「口コミ」と同じ役割を果たしています。

この記事を読むことで、Instagram特有のクレーム対応方法、コメント削除・ブロックの判断基準、炎上拡大を防ぐ初動対応が分かります。

## Instagramにおける「口コミ」の種類

Instagramには公式のレビュー機能はありませんが、以下の場所で「口コミ」が発生します。

### 1. 投稿へのコメント

企業アカウントの投稿に対して、顧客がコメントを残します。

**特徴**:
- 公開されているため、第三者も閲覧可能
- 削除・非表示が可能（アカウント所有者のみ）
- 返信すると、コメント欄で会話が続く

**リスク**:
- ネガティブコメントが放置されると、他のフォロワーにも見られる
- コメント欄で言い合いになると、炎上しやすい

### 2. DM（ダイレクトメッセージ）

顧客から非公開でメッセージが送られます。

**特徴**:
- 非公開のため、第三者には見えない
- 返信は1対1のやり取り
- 既読・未読が表示される

**リスク**:
- 対応が遅いと「無視された」と公開の場で晒される
- 不適切な返信をすると、スクリーンショットで拡散される

### 3. ストーリーズメンション

ストーリーズで「@アカウント名」とタグ付けされます。

**特徴**:
- 24時間で消える
- メンションされると通知が来る
- リポスト（再シェア）が可能

**リスク**:
- ネガティブなメンションが拡散されると、フォロワーに見られる
- スクリーンショットで保存され、永久に残る可能性がある

### 4. ハッシュタグ投稿

「#店舗名」などのハッシュタグで、顧客が独自に投稿します。

**特徴**:
- 企業アカウントとは無関係に投稿される
- 削除・編集は不可能（投稿者のみ可能）
- ハッシュタグ検索で誰でも閲覧可能

**リスク**:
- ネガティブなハッシュタグ投稿が増えると、ブランドイメージが悪化
- 削除できないため、対応が難しい

## コメント欄への返信テンプレート

Instagram投稿のコメント欄は公開されているため、返信内容が第三者にも見られます。

### 返信テンプレート（ネガティブコメント・基本パターン）

```
コメントありがとうございます。

ご不快な思いをさせてしまい、誠に申し訳ございません。
詳しくお話をお伺いしたいので、お手数ですがDMをお送りいただけますでしょうか？

誠意をもって対応させていただきます。

[アカウント名]
```

**ポイント**:
- 公開の場では簡潔に謝罪のみ
- 詳細なやり取りはDMに誘導
- 第三者に「誠実に対応している」と見せる

### 返信テンプレート（誤解・事実誤認のコメント）

```
コメントありがとうございます。

ご指摘いただいた点につきまして、誤解があるようです。
当店では〇〇のサービスを提供しており、〇〇の対応を行っております。

もし詳しくご説明が必要でしたら、DMでお問い合わせください。

[アカウント名]
```

**ポイント**:
- 事実を簡潔に訂正
- 攻撃的にならない
- 長文の説明は避ける

### 返信テンプレート（インフルエンサーからのネガティブコメント）

```
〇〇様、コメントありがとうございます。

ご期待に添えず、誠に申し訳ございませんでした。
いただいたご意見を真摯に受け止め、改善に努めてまいります。

もしよろしければ、DMで詳しくお話をお伺いできますでしょうか？

[アカウント名]
```

**ポイント**:
- インフルエンサーには丁寧に対応（炎上拡大を防ぐため）
- 公開の場で反論しない
- フォロワー数が多い場合は、弁護士に相談してからDM対応

## 絶対に書いてはいけないNGコメント返信

Instagram のコメント返信には、炎上リスクの高いNG表現があります。

### NG例1: 顧客を批判する

```
「あなたの使い方が間違っています。説明書を読んでください」
```

**リスク**: 上から目線と受け取られ、炎上します。他のフォロワーからも「感じ悪い」と批判されます。

### NG例2: 他の顧客を引き合いに出す

```
「他のお客様からは高評価をいただいております。おそらくあなただけの問題です」
```

**リスク**: 「あなただけが変」と言っているように聞こえ、レビュアーを刺激します。

### NG例3: 長文の言い訳

```
「当店は〇〇にこだわっており、〇〇の食材を使用し、〇〇の調理法で…（300文字以上の説明）」
```

**リスク**: 「言い訳ばかり」と受け取られ、スクリーンショットで晒されます。

### NG例4: 感情的に反論する

```
「そんなことは言っていません！嘘を書かないでください！」
```

**リスク**: 感情的な返信は、必ず炎上します。冷静さを失うと、さらに批判されます。

## DMでの対応方法

DMは非公開ですが、スクリーンショットで晒されるリスクがあります。

### DM返信の基本ルール

1. **必ず丁寧語を使う**: タメ口は絶対NG
2. **謝罪から始める**: 事実確認の前に、まず謝罪
3. **長文を避ける**: 1メッセージ200文字以内に収める
4. **スクリーンショット前提で書く**: 晒されても問題ない内容のみ書く

### DM返信テンプレート（クレーム対応）

```
お問い合わせいただき、ありがとうございます。

この度はご不快な思いをさせてしまい、誠に申し訳ございませんでした。

詳しくお話をお伺いしたいので、以下の点について教えていただけますでしょうか？

・ご来店日時
・ご注文内容
・具体的にどのような点が不満だったか

お手数をおかけいたしますが、よろしくお願いいたします。

[アカウント名]
```

### DM返信テンプレート（返金・補償対応）

```
ご連絡いただき、ありがとうございます。

この度は誠に申し訳ございませんでした。

つきましては、以下の対応をさせていただきたく存じます。

【対応内容】
1. 全額返金
2. 代替商品の提供
3. 次回ご利用時の割引クーポン

ご希望の対応方法をお知らせいただけますでしょうか？

[アカウント名]
```

**重要**: DMで補償を提示する際は、「投稿削除と引き換え」と読まれないように注意してください。

## コメント削除・ブロックの判断基準

Instagramでは、コメントの削除やアカウントのブロックが可能ですが、判断を誤ると逆効果になります。

### 削除すべきコメント

以下のコメントは、削除しても問題ありません。

| コメント内容 | 削除理由 | リスク |
|------------|---------|--------|
| 誹謗中傷・差別的表現 | Instagramコミュニティガイドライン違反 | 低 |
| スパム・広告 | 営業妨害 | 低 |
| 個人情報の暴露 | プライバシー侵害 | 低 |
| 脅迫・恐喝 | 犯罪行為 | 低（警察に通報も検討） |

### 削除すべきでないコメント

以下のコメントを削除すると、「都合の悪いコメントを消している」と批判されます。

| コメント内容 | 削除しない理由 | 代わりの対応 |
|------------|---------------|-------------|
| 接客への不満 | 正当なクレーム | 返信で謝罪・改善策を示す |
| 料理への不満 | 主観的な意見 | 返信で「ご期待に添えず申し訳ございません」 |
| 価格への不満 | 正当な意見 | 返信で価値を説明 |

### ブロックすべきアカウント

以下のアカウントは、ブロックしても問題ありません。

- 繰り返し誹謗中傷を投稿するアカウント
- スパムアカウント
- 競合店による嫌がらせアカウント（証拠がある場合）

**ブロックの注意点**:
- ブロックすると、相手に通知は行きませんが、アカウントが表示されなくなるため、ブロックされたことに気づかれます
- ブロックされたことに怒り、他のアカウントで攻撃してくる可能性があります

## ストーリーズメンションへの対応

ストーリーズでネガティブなメンションを受けた場合の対応方法です。

### ストーリーズメンションの種類

| 種類 | 説明 | 対応 |
|------|------|------|
| @メンション | 自アカウントがタグ付けされる | リポスト可能 |
| ハッシュタグメンション | #店舗名などで言及される | リポスト不可、削除不可 |
| 位置情報タグ | 店舗の位置情報が付けられる | 削除不可 |

### ネガティブメンションへの対応

**対応1: リポストしない**
- ネガティブなメンションは、絶対にリポストしない
- リポストすると、自分のフォロワーにも拡散されてしまう

**対応2: DMで謝罪**
- メンション投稿者にDMを送り、謝罪と改善策を伝える

**対応3: 24時間待つ**
- ストーリーズは24時間で消えるため、過剰に反応しない
- ただし、スクリーンショットで保存されるリスクはある

### ストーリーズDM返信テンプレート

```
ストーリーズでのご投稿、拝見いたしました。

この度はご不快な思いをさせてしまい、誠に申し訳ございませんでした。

もしよろしければ、詳しくお話をお伺いしたいので、ご連絡いただけますでしょうか？

誠意をもって対応させていただきます。

[アカウント名]
```

## インフルエンサーからのネガティブ投稿への対応

フォロワー数の多いインフルエンサーから低評価を受けた場合、拡散力が大きいため、慎重な対応が必要です。

### インフルエンサー対応の基本方針

1. **すぐに反応しない**: 感情的な返信は炎上リスクが高い
2. **弁護士に相談する**: フォロワー1万人以上の場合、法的リスクを確認
3. **DMで誠実に対応**: 公開の場で反論しない
4. **補償を検討**: 金額よりも、誠意を見せることが重要

### インフルエンサーへのDM返信テンプレート

```
〇〇様

いつも当店をご利用いただき、ありがとうございます。

この度はご期待に添えず、誠に申し訳ございませんでした。
ご指摘いただいた点につきまして、スタッフ一同で共有し、改善に取り組んでおります。

つきましては、以下の対応をさせていただきたく存じます。

【対応内容】
- 全額返金
- 次回ご来店時のVIP対応
- 店長からの直接謝罪

お忙しいところ恐れ入りますが、ご都合の良い日時をお知らせいただけますでしょうか？

[店舗名] 店長
```

**重要**: 「投稿を削除してほしい」とは書かないでください。レビュー操作と疑われます。

## Instagramでの炎上を防ぐ予防策

炎上を未然に防ぐための日常的な運用方法です。

### 予防策1: コメント欄を定期的に監視

毎日、以下のタイミングでコメント欄をチェックします。
- 朝9時
- 昼12時
- 夕方17時
- 夜21時

**目的**: ネガティブコメントを早期発見し、初動対応を早める

### 予防策2: コメントフィルター機能を活用

Instagram の「コメントフィルター」機能で、特定の単語を含むコメントを自動的に非表示にします。

**設定方法**:
1. プロフィール画面右上の「≡」→「設定とプライバシー」
2. 「非表示の単語」を選択
3. フィルターしたい単語を追加

**フィルター推奨ワード**:
- 誹謗中傷系: 「死ね」「バカ」「クズ」など
- 競合誘導系: 「〇〇店の方が良い」など

### 予防策3: ハッシュタグを監視

定期的に「#店舗名」で検索し、ネガティブ投稿がないか確認します。

**監視ツール**:
- Instagram公式のハッシュタグ検索
- SNS監視ツール（Brandwatch、Hootsuite等）

### 予防策4: ポジティブ投稿をリポスト

高評価の投稿を積極的にリポストすることで、ネガティブ投稿を埋もれさせます。

**リポスト方法**:
1. ストーリーズでリポスト
2. フィード投稿でユーザー投稿を引用
3. 「お客様の声」としてハイライトに保存
