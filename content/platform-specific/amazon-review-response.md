## この記事は誰のため？

この記事は、Amazonで★1〜★2の低評価カスタマーレビューを書かれてしまったEC事業者・出品者に向けて書かれています。

Amazonのレビューシステムは、Googleマップや食べログとは全く異なるルールがあります。「出品者コメント」は投稿できるが返信機能とは異なる、削除依頼は非常に厳しい、購入確認済み（Verified Purchase）のレビューは削除されにくいなど、独自の特徴があります。

この記事を読むことで、Amazonカスタマーレビューへの正しい対応方法、削除依頼の手順、出品者コメントの書き方、ポリシー違反の判断基準が分かります。

## Amazonレビューの基本ルール（他プラットフォームとの違い）

Amazonのカスタマーレビューシステムには、他のプラットフォームにはない独自のルールがあります。

### 「返信」ではなく「出品者コメント」

Amazonには、GoogleマップやY食べログのような「返信機能」はありません。代わりに「出品者コメント」という機能があります。

| 機能 | Googleマップ | 食べログ | Amazon |
|------|-------------|---------|--------|
| 返信機能 | ✅ あり | ✅ あり（有料会員のみ） | ❌ なし |
| 出品者コメント | ❌ なし | ❌ なし | ✅ あり |
| 投稿後の編集 | ✅ 可能 | ✅ 可能 | ❌ 投稿後60日間のみ |
| レビュアーへの通知 | ✅ あり | ✅ あり | ❌ なし |

**重要**: 出品者コメントを投稿しても、レビュアーに通知は行きません。コメントはレビューページに表示されますが、レビュアー本人が再訪問しない限り気づかれません。

### 購入確認済み（Verified Purchase）の重要性

Amazonでは、実際に商品を購入した顧客のレビューには「Amazonで購入」というバッジが付きます。

**購入確認済みレビューの特徴**:
- 削除依頼がほとんど承認されない
- 検索結果で上位に表示される
- 顧客から信頼されやすい

**購入確認なしレビューの特徴**:
- 削除依頼が承認されやすい
- 検索結果で下位に表示される
- 「サクラレビュー」と疑われやすい

### レビュー削除の厳しさ

Amazonのレビュー削除基準は、他のプラットフォームよりも厳しいと言われています。

**削除が承認されやすいケース**:
- 購入していない人のレビュー（Verified Purchaseなし）
- 競合商品への誘導リンクがある
- 明らかな誹謗中傷・差別的表現
- 個人情報が含まれている

**削除が承認されにくいケース**:
- 購入確認済みの低評価レビュー
- 主観的な不満（「期待外れ」「使いにくい」など）
- 配送遅延のクレーム（Amazon側の問題でも、商品レビューに書かれると削除されにくい）

## 出品者コメントの書き方（返信テンプレート）

Amazonの出品者コメントは、レビューへの「返信」ではなく、第三者（将来の顧客）に向けた説明として書きます。

### 出品者コメントのテンプレート（基本パターン）

```
ご購入いただき、誠にありがとうございます。
また、貴重なご意見をいただき感謝申し上げます。

ご指摘いただいた点につきまして、以下の通り対応させていただきます。

【改善内容】
- 〇〇の仕様を見直し、次回ロットから改良版を出荷予定です
- 商品ページに〇〇の注意事項を追加記載いたしました

今後もお客様の声を真摯に受け止め、商品改善に努めてまいります。

[店舗名/ブランド名]
```

### 出品者コメントのテンプレート（配送遅延クレーム）

```
この度は配送にお時間をいただき、誠に申し訳ございませんでした。

配送遅延につきましては、配送業者および当店の在庫管理体制を見直し、今後このようなことがないよう改善いたします。

なお、商品の品質には問題ございませんので、ご安心くださいませ。

もしご不明点がございましたら、Amazonメッセージ機能よりお問い合わせください。

[店舗名/ブランド名]
```

### 出品者コメントのテンプレート（誤解・使い方の問題）

```
ご購入いただき、ありがとうございます。

ご指摘いただいた点につきまして、商品説明が不十分であったこと、お詫び申し上げます。

本商品は〇〇の用途を想定して設計されており、〇〇の使い方をしていただくことで本来の性能を発揮いたします。

商品ページに詳しい使用方法を追加記載いたしましたので、もしよろしければご確認くださいませ。

[店舗名/ブランド名]
```

## 絶対に書いてはいけないNG出品者コメント

Amazonの出品者コメントには、厳格なルールがあり、違反するとコメントが削除されるだけでなく、アカウント停止のリスクもあります。

### NG例1: レビュアーに直接反論する

```
「あなたの使い方が間違っています。説明書をよく読んでください」
```

**リスク**: 攻撃的な表現は、Amazonのコミュニティガイドライン違反です。出品者コメントが削除され、アカウント警告を受ける可能性があります。

### NG例2: 高評価を依頼する

```
「もしご満足いただけましたら、★5の評価に変更していただけますと幸いです」
```

**リスク**: レビュー操作とみなされ、アカウント停止のリスクがあります。Amazonは評価変更の依頼を厳しく禁止しています。

### NG例3: 連絡先を記載する

```
「詳しくは当店のメールアドレス(example@example.com)までご連絡ください」
```

**リスク**: Amazon外への誘導は禁止されています。出品者コメントが削除されます。Amazonメッセージ機能を使うように促すのは問題ありません。

### NG例4: 他の商品へのリンクを貼る

```
「当店の別商品『〇〇』の方が、お客様のニーズに合っているかもしれません」
```

**リスク**: レビューページでの営業行為は禁止されています。出品者コメントが削除されます。

### NG例5: 金銭的な補償を提示する

```
「返金いたしますので、Amazonメッセージでご連絡ください」
```

**リスク**: レビュー削除と引き換えの返金と疑われ、レビュー操作とみなされます。返金対応は、Amazon公式の返品システムを通じて行う必要があります。

## Amazonレビューの削除依頼手順

Amazonのレビュー削除依頼は、「カスタマーレビューガイドライン」に違反している場合のみ承認されます。

### 削除依頼の手順

#### ステップ1: セラーセントラルにログイン

Amazonセラーセントラル（出品者用管理画面）にログインします。

#### ステップ2: 「パフォーマンス」→「カスタマーレビュー」を選択

削除したいレビューを表示します。

#### ステップ3: 「不正使用を報告」をクリック

レビューの右側にある「不正使用を報告」ボタンをクリックします。

#### ステップ4: 違反理由を選択

以下の違反理由から選択します。

| 違反理由 | 具体例 | 承認可能性 |
|---------|--------|----------|
| 購入していない商品のレビュー | Verified Purchaseなしで投稿 | 高 |
| 誹謗中傷・脅迫 | 「詐欺」「訴える」など | 高 |
| 個人情報が含まれている | 出品者の住所・電話番号など | 高 |
| 商品とは無関係な内容 | 配送業者への不満のみ | 中 |
| 競合商品への誘導 | 「〇〇の方が良い」とリンク | 高 |
| 報酬と引き換えのレビュー | 「無料でもらった」と記載 | 高 |

#### ステップ5: 詳細な説明を記入

なぜそのレビューがガイドライン違反にあたるのか、具体的に説明します。

**良い申請文の例**:
```
このレビューは、購入確認済みバッジがなく、実際に商品を購入していない可能性が高いです。また、レビュー内容が当店の商品ではなく、競合商品の説明になっており、「〇〇の商品の方が良い」と他商品へ誘導する内容が含まれています。

Amazonカスタマーレビューガイドラインの「購入していない商品のレビュー」および「他の商品への誘導」に違反していると判断されるため、削除をお願いいたします。
```

**悪い申請文の例**:
```
不当な低評価レビューです。削除してください。
```

### 削除依頼の承認率を高めるポイント

1. **ガイドライン違反を明確に指摘する**: 「〇〇ガイドラインの〇〇条に違反」と具体的に書く
2. **客観的な証拠を提示する**: スクリーンショット、購入履歴の確認など
3. **感情的にならない**: 「ひどい」「許せない」などの表現は避ける
4. **複数のレビューをまとめて申請しない**: 1件ずつ、丁寧に申請する

## 購入者からの問い合わせへの対応

低評価レビューを投稿した顧客から、Amazonメッセージで問い合わせが来る場合があります。

### Amazonメッセージでの返信ルール

Amazonメッセージには、厳格なルールがあります。

**やって良いこと**:
- 商品の使い方を説明する
- 返品・交換の手続きを案内する
- 不良品の対応を行う

**やってはいけないこと**:
- レビュー評価の変更を依頼する
- 返金と引き換えにレビュー削除を依頼する
- Amazon外の連絡先を教える
- 高評価レビューを書くように依頼する

### 返信テンプレート（Amazonメッセージ用）

```
件名: ご購入いただいた商品について

〇〇様

この度は当店の商品をご購入いただき、誠にありがとうございます。

レビューにてご指摘いただいた点につきまして、大変申し訳ございませんでした。

つきましては、以下の対応をさせていただきたく存じます。

【対応内容】
1. 商品の返品・交換（Amazonの返品システムをご利用ください）
2. 使用方法の詳しいご説明（添付PDFをご確認ください）
3. 代替商品のご提案

もしご不明点がございましたら、お気軽にお問い合わせくださいませ。

[店舗名/ブランド名]
```

## 低評価レビューを予防する方法

Amazonで低評価レビューを減らすには、以下の予防策が効果的です。

### 予防策1: 商品ページの説明を充実させる

レビューで「思っていたのと違う」というクレームが多い場合、商品説明が不十分な可能性があります。

**改善ポイント**:
- 商品サイズを具体的に記載（cm単位で）
- 使用シーンの写真を複数掲載
- 「よくある質問」セクションを追加
- 注意事項を目立つ位置に記載

### 予防策2: 早期警戒レビュー（Early Reviewer Program）を活用

Amazonの「早期警戒レビュー」プログラムに参加すると、初期のレビューを獲得しやすくなります。

**メリット**:
- 新商品でも早くレビューが集まる
- ★4〜★5の高評価が付きやすい

**デメリット**:
- 費用がかかる（1商品あたり$60）
- 必ずしも高評価とは限らない

### 予防策3: パッケージに使用方法を同梱

商品の使い方が分かりにくい場合、「使えない」という低評価レビューが付きます。

**対策**:
- 日本語の取扱説明書を同梱
- QRコードで動画マニュアルに誘導
- 「困ったときは」の問い合わせ先を記載

### 予防策4: フォローアップメールを送る

Amazon公式の「購入者へのメッセージ」機能を使い、フォローアップメールを送ります。

**送って良い内容**:
- 商品到着の確認
- 使用方法の案内
- 不具合があった場合の連絡先

**送ってはいけない内容**:
- レビュー投稿の依頼
- ★5評価の依頼
- クーポンと引き換えのレビュー依頼

## 配送遅延レビューへの対応

Amazonでは、配送遅延のクレームが商品レビューに書かれることがあります。

### 配送遅延レビューの削除は難しい

「配送が遅かった」というレビューは、商品とは無関係ですが、Amazonは削除申請を承認しないことが多いです。

**理由**:
- Amazonは「配送も含めた購入体験全体」をレビュー対象としている
- FBA（フルフィルメント by Amazon）を使っていても、出品者の責任とみなされる

### 配送遅延レビューへの出品者コメント例

```
この度は配送にお時間をいただき、誠に申し訳ございませんでした。

配送遅延につきましては、配送業者との連携を強化し、今後このようなことがないよう改善いたします。

商品の品質には問題ございませんので、安心してご使用くださいませ。

[店舗名/ブランド名]
```
