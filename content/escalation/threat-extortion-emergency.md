## この記事は誰のため？

この記事は、Googleマップや口コミサイトで「金を払わないと低評価を消さない」「訴えてやる」「店を潰す」などの脅迫・恐喝的なレビューを受けた店舗経営者に向けて書かれています。

脅迫・恐喝は刑法違反であり、警察への被害届提出や刑事告訴が可能です。しかし、対応を誤ると証拠が消されたり、逆に名誉毀損で訴えられるリスクがあります。

この記事を読むことで、脅迫レビューへの緊急対応手順、警察・弁護士との連携方法、証拠保全の仕方、削除申請の流れが分かります。

## 脅迫・恐喝レビューとは？

脅迫・恐喝レビューは、以下のような内容を含む口コミです。

### 脅迫罪（刑法222条）に該当する可能性がある表現

**脅迫の定義**: 生命、身体、自由、名誉または財産に対し害を加える旨を告知すること

**具体例**:
- 「このレビューを消したければ金を払え」
- 「店を潰してやる」
- 「経営者の自宅を晒してやる」
- 「お前の家族に危害を加える」
- 「従業員を全員クビにしてやる」

**法定刑**: 2年以下の懲役または30万円以下の罰金

### 恐喝罪（刑法249条）に該当する可能性がある表現

**恐喝の定義**: 人を恐喝して財物を交付させること、または財産上不法の利益を得ること

**具体例**:
- 「10万円払えば低評価を消してやる」
- 「謝罪料として5万円用意しろ」
- 「示談金を払わないとネットで拡散する」
- 「無料で商品をよこせ、さもないと悪評を広める」

**法定刑**: 10年以下の懲役

### 業務妨害罪（刑法233条）に該当する可能性がある表現

**偽計業務妨害の定義**: 虚偽の風説を流布し、または偽計を用いて、人の業務を妨害すること

**具体例**:
- 「この店は食中毒を隠蔽している」（虚偽の場合）
- 「この店は不法移民を雇っている」（虚偽の場合）
- 「保健所から営業停止命令が出ている」（虚偽の場合）

**法定刑**: 3年以下の懲役または50万円以下の罰金

## 緊急対応フローチャート

脅迫・恐喝レビューを受けた場合、以下の順序で対応します。

### ステップ1: 証拠保全（最優先）

**即座に実施すべきこと**:

1. **スクリーンショット撮影**
   - レビュー全文
   - 投稿者のアカウント情報
   - 投稿日時
   - URLも含めて撮影

2. **ウェブ魚拓（archive.org等）に保存**
   - https://archive.org/web/
   - https://archive.md/
   - 削除されても証拠が残る

3. **DM・メールがある場合は全て保存**
   - ヘッダー情報も含めて保存
   - PDFで出力して保管

**重要**: 相手に気づかれる前に証拠を保全してください。返信や削除申請をすると、相手が投稿を消す可能性があります。

### ステップ2: 警察への相談・被害届提出

**相談先**:
- 所轄の警察署（生活安全課）
- サイバー犯罪相談窓口（都道府県警察本部）
- 警察相談専用電話 #9110

**持参すべき証拠**:
- スクリーンショット（印刷したもの）
- ウェブ魚拓のURL
- DM・メールの記録
- 被害状況をまとめたメモ

**被害届に記載する内容**:
- いつ（投稿日時）
- どこで（Googleマップ、食べログ等のプラットフォーム）
- 誰が（投稿者のアカウント名・ID）
- 何をした（脅迫・恐喝の具体的内容）
- 被害状況（売上減少、精神的苦痛等）

**警察の対応**:
- 脅迫・恐喝の証拠が明確な場合 → 被害届受理、捜査開始
- 証拠が不十分な場合 → 相談扱い、様子見
- 民事不介入と判断された場合 → 弁護士への相談を勧められる

### ステップ3: 弁護士への相談

**相談すべきタイミング**:
- 警察が動いてくれない場合
- 金銭要求が具体的な場合
- 民事訴訟を検討する場合

**弁護士の選び方**:
- インターネット法務に詳しい弁護士
- 刑事告訴の経験がある弁護士
- 削除請求（発信者情報開示請求）の実績がある弁護士

**相談料の目安**:
- 初回相談: 30分5,000円〜1万円（無料相談もあり）
- 削除請求: 10万円〜30万円
- 発信者情報開示請求: 50万円〜100万円
- 刑事告訴状の作成: 20万円〜50万円

### ステップ4: プラットフォームへの削除申請

**Googleマップの場合**:

1. Google マイビジネスにログイン
2. 該当レビューを選択
3. 「不適切なクチコミとして報告」をクリック
4. 報告理由: 「嫌がらせや暴力的な言葉」を選択

**削除が認められやすいケース**:
- 明確な脅迫・恐喝表現がある
- 金銭要求が書かれている
- 個人情報（住所・電話番号等）が晒されている

**削除が認められにくいケース**:
- 単なる悪口・不満（「最悪の店」等）
- 主観的な評価（「まずい」「高い」等）

**削除までの期間**: 通常3日〜2週間（緊急性が高い場合は早まることも）

### ステップ5: 発信者情報開示請求（必要に応じて）

プラットフォームに投稿者の情報開示を求める法的手続きです。

**手順**:
1. プラットフォーム（Google等）に仮処分申請
2. IPアドレス・タイムスタンプを取得
3. インターネットプロバイダ（ISP）に本開示請求
4. 投稿者の氏名・住所を特定
5. 民事訴訟（損害賠償請求）または刑事告訴

**費用**: 50万円〜100万円（弁護士費用含む）
**期間**: 6ヶ月〜1年

## 絶対にやってはいけないNG対応

脅迫・恐喝レビューへの対応を誤ると、証拠が消されたり、逆に訴えられるリスクがあります。

### NG対応1: 金銭を支払う

```
相手の要求: 「10万円払えば低評価を消してやる」
NG対応: 「分かりました。お支払いします」
```

**リスク**:
- 恐喝に応じると、さらに要求がエスカレートする
- 支払った事実が証拠となり、「示談が成立した」と主張される
- 他の恐喝犯を呼び込むリスク

**正しい対応**:
- 一切の金銭要求に応じない
- 警察・弁護士に相談
- 証拠を保全して削除申請

### NG対応2: 公開返信で反論する

```
NG返信: 「あなたの書き込みは脅迫罪に該当します。警察に通報しました」
```

**リスク**:
- 相手に警戒され、投稿が削除される（証拠が消える）
- 「逆ギレされた」とSNSで拡散される
- 公開の場での法的主張は、逆に名誉毀損で訴えられる可能性

**正しい対応**:
- 公開返信は一切しない
- 静かに証拠を保全し、警察・弁護士に相談

### NG対応3: DMで脅し返す

```
NG対応: 「お前を訴えてやる」「警察に行く」とDMで送る
```

**リスク**:
- 逆に脅迫罪で訴えられる可能性
- 感情的なやり取りが証拠として残り、不利になる

**正しい対応**:
- 相手と直接やり取りしない
- 弁護士を通じて連絡

### NG対応4: 自力で投稿者を特定しようとする

```
NG対応: 「IPアドレスから住所を特定してやる」
```

**リスク**:
- 不正アクセス禁止法違反（刑法168条の2）
- 住居侵入罪、ストーカー規制法違反のリスク

**正しい対応**:
- 発信者情報開示請求は弁護士に依頼

## 公開返信テンプレート（脅迫レビュー用）

脅迫・恐喝レビューには、**公開返信をしないことが原則**です。しかし、他の閲覧者への説明が必要な場合は、以下のテンプレートを参考にしてください。

### テンプレート1: 警察に相談済み（軽度の脅迫）

```
この度はご意見をいただき、ありがとうございます。

お客様の投稿内容につきまして、当店としましては事実と異なる点がございます。

詳しくお話をお伺いしたく、恐れ入りますが店舗まで直接ご連絡いただけますでしょうか。

[店舗名] 店長
```

**ポイント**:
- 「脅迫」「警察」などの言葉は使わない
- 淡々と対応を伝える
- 他の閲覧者に「店側は冷静に対応している」と印象づける

### テンプレート2: 弁護士に相談済み（重度の恐喝）

```
この度はご連絡いただき、ありがとうございます。

お客様からのご要望につきまして、弁護士と相談の上、適切に対応させていただきます。

今後のやり取りは、弁護士を通じて行わせていただきますので、ご了承ください。

[店舗名] 店長
```

**ポイント**:
- 「弁護士」という言葉で、法的対応を示唆
- 直接のやり取りを拒否
- 感情的にならず、淡々と対応

### テンプレート3: 返信しない（最も推奨）

脅迫・恐喝レビューには、**返信しないことが最善**です。

**理由**:
- 返信すると、相手が投稿を削除する可能性（証拠が消える）
- 公開の場での法的主張は、逆に名誉毀損のリスク
- 他の閲覧者は、投稿内容を見れば「異常な投稿」と判断できる

## 予防策: 脅迫・恐喝レビューを受けないために

脅迫・恐喝レビューを完全に防ぐことはできませんが、リスクを減らす対策はあります。

### 予防策1: 顧客対応を記録する

**記録すべき内容**:
- 接客時の会話内容
- クレーム対応の経緯
- 返金・交換の記録
- 防犯カメラの映像

**効果**:
- 脅迫レビューが書かれても、事実確認ができる
- 警察・弁護士への説明がスムーズ

### 予防策2: クレーム対応マニュアルを整備

**マニュアルに含めるべき内容**:
- クレームの種類別対応方法
- エスカレーションルール（店長・本部への報告基準）
- 脅迫・恐喝を受けた場合の対応フロー

**効果**:
- スタッフが冷静に対応できる
- 感情的な対応を避け、脅迫を誘発しない

### 予防策3: 口コミ監視ツールの導入

脅迫レビューは、投稿後すぐに証拠保全する必要があります。

**推奨ツール**:
- Googleアラート（無料）
- レピュ研（月額2万円〜）
- クチコミル（月額1万円〜）

**効果**:
- 投稿されたらすぐに通知
- 証拠保全が迅速にできる

### 予防策4: 顧問弁護士との契約

脅迫・恐喝レビューは、スピードが重要です。

**顧問契約のメリット**:
- すぐに相談できる
- 削除申請・発信者情報開示請求がスムーズ
- 抑止力になる（「顧問弁護士がいる」と明記）

**費用**: 月額3万円〜5万円

## 脅迫レビューの判例・事例

実際に刑事告訴・民事訴訟に発展した事例です。

### 事例1: 飲食店への恐喝レビュー（逮捕）

**概要**:
- 飲食店に「10万円払わないと悪評を書く」と脅迫
- 店側が警察に被害届を提出
- 投稿者が恐喝未遂罪で逮捕

**判決**: 懲役1年（執行猶予3年）

**教訓**: 金銭要求は明確な恐喝罪。警察は動く。

### 事例2: 美容クリニックへの脅迫レビュー（民事訴訟）

**概要**:
- 「施術ミスで顔が崩れた。慰謝料100万円払え」と投稿
- クリニック側が発信者情報開示請求
- 投稿者を特定し、名誉毀損で損害賠償請求

**判決**: 投稿者に50万円の賠償命令

**教訓**: 虚偽の投稿は、名誉毀損で訴えられる。

### 事例3: ホテルへの業務妨害レビュー（刑事告訴）

**概要**:
- 「このホテルは盗撮カメラが仕掛けられている」と虚偽投稿
- 予約キャンセルが相次ぎ、売上が激減
- ホテル側が偽計業務妨害罪で刑事告訴

**判決**: 投稿者に罰金30万円

**教訓**: 虚偽の風説による業務妨害は、刑事罰の対象。
