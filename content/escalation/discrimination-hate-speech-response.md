## この記事は誰のため?

この記事は、Googleマップや口コミサイトで人種差別、性差別、障害者差別などのヘイトスピーチを含むレビューを受けた店舗経営者・個人事業主に向けて書かれています。

差別的発言は、名誉毀損罪・侮辱罪に該当する可能性があり、プラットフォームの削除対象にもなります。しかし、削除基準は曖昧で、申請しても却下されるケースがあります。

この記事を読むことで、差別的口コミの削除申請方法、法的措置の判断基準、人権擁護委員会への相談方法が分かります。

## 差別的発言・ヘイトスピーチとは？

### ヘイトスピーチの定義

ヘイトスピーチとは、人種、民族、国籍、性別、性的指向、障害、宗教などを理由に、特定の個人や集団を攻撃・侮辱する表現のことです。

**法律上の定義**:
- 「ヘイトスピーチ解消法」（本邦外出身者に対する不当な差別的言動の解消に向けた取組の推進に関する法律）2016年施行
- 刑法230条（名誉毀損罪）、刑法231条（侮辱罪）

### 差別的口コミの種類

#### 1. 人種・民族差別

**具体例**:
- 「この店は外国人がやっている。不潔だ」
- 「中国人の店だから信用できない」
- 「店員が韓国人。日本語が下手で不快」

**削除基準**:
- Googleマップ: 「人種や民族を理由にした攻撃的な内容」は削除対象
- 食べログ: 「差別的な表現」は削除対象

#### 2. 性差別・性的指向差別

**具体例**:
- 「女の店長だから仕事ができない」
- 「ゲイの店員がいて気持ち悪い」
- 「LGBTを雇うな。子供に悪影響だ」

**削除基準**:
- Googleマップ: 「性別や性的指向を理由にした攻撃的な内容」は削除対象
- 食べログ: 「性的指向に関する差別的表現」は削除対象

#### 3. 障害者差別

**具体例**:
- 「車椅子の客がいて邪魔だった」
- 「知的障害者を雇うな。危ない」
- 「身体障害者が働いていて気分が悪い」

**削除基準**:
- Googleマップ: 「障害を理由にした攻撃的な内容」は削除対象
- 障害者差別解消法違反の可能性あり

#### 4. 年齢差別

**具体例**:
- 「老人ばかりで動きが遅い」
- 「若い店員は生意気だ」

**削除基準**:
- 単なる年齢への言及は削除されにくい
- 「老害」「ガキ」など侮辱的表現があれば削除される可能性

#### 5. 宗教差別

**具体例**:
- 「イスラム教徒がやっている店。テロリストかも」
- 「宗教団体が経営している。洗脳されそう」

**削除基準**:
- Googleマップ: 「宗教を理由にした攻撃的な内容」は削除対象
- 特定の宗教への誹謗中傷は削除されやすい

## 差別的口コミの法的評価

差別的発言が、刑法上の犯罪に該当するかの判断基準です。

### 名誉毀損罪（刑法230条）の成立要件

**要件**:
1. 公然と（不特定多数が閲覧できる場所で）
2. 事実を摘示し（具体的な事実を述べ）
3. 人の名誉を毀損した

**具体例**:
- 「この店の店長は前科者だ」（虚偽の場合）
- 「この店は脱税している」（虚偽の場合）

**法定刑**: 3年以下の懲役もしくは禁錮または50万円以下の罰金

**差別的発言との関係**:
- 単なる差別的な感想（「外国人は嫌い」等）は、名誉毀損に該当しない
- 具体的な事実を摘示した場合（「この店は不法移民を雇っている」等）は名誉毀損になる可能性

### 侮辱罪（刑法231条）の成立要件

**要件**:
1. 事実を摘示せずに
2. 公然と
3. 人を侮辱した

**具体例**:
- 「バカ」「クズ」「ゴミ」
- 「死ね」「消えろ」

**法定刑**: 1年以下の懲役もしくは禁錮もしくは30万円以下の罰金または拘留もしくは科料
※2022年の刑法改正で厳罰化

**差別的発言との関係**:
- 「〇〇人は汚い」「障害者は気持ち悪い」などの侮辱的表現は、侮辱罪に該当する可能性

### ヘイトスピーチ解消法の限界

**重要**: ヘイトスピーチ解消法には罰則規定がありません。

**法律の内容**:
- 国や地方自治体に、ヘイトスピーチ解消のための啓発・教育を求める理念法
- 違反者への刑事罰はない

**実務上の効果**:
- プラットフォーム（Google、食べログ等）が、削除基準を設ける根拠になる
- 民事訴訟（損害賠償請求）の際に、違法性の根拠になる

## 削除申請の手順

差別的口コミは、プラットフォームの削除対象になります。

### Googleマップでの削除申請

**手順**:
1. Google マイビジネスにログイン
2. 該当レビューを選択
3. 「不適切なクチコミとして報告」をクリック
4. 報告理由: 「嫌がらせや暴力的な言葉」または「差別的な内容」を選択

**削除されやすいケース**:
- 人種・民族を理由にした明確な攻撃的表現
- 性的指向・障害を理由にした侮辱的表現
- 特定の宗教への誹謗中傷

**削除されにくいケース**:
- 単なる不満（「サービスが悪い」等）
- 曖昧な表現（「なんとなく気持ち悪い」等）

**削除までの期間**: 通常3日〜2週間

### 食べログでの削除申請

**手順**:
1. 食べログの「口コミに関するお問い合わせ」フォームから申請
2. 該当口コミのURL、削除理由を記載

**削除基準**:
- 差別的な表現
- 誹謗中傷
- プライバシー侵害

**削除までの期間**: 通常1週間〜1ヶ月

### 削除申請が却下された場合

**再申請**:
- より具体的な削除理由を記載
- 「差別的である理由」を詳しく説明
- 該当する法律（名誉毀損罪、侮辱罪等）を引用

**法的措置**:
- 弁護士に依頼し、発信者情報開示請求
- 裁判所を通じて削除仮処分申請

## 公開返信テンプレート（差別的口コミ用）

差別的口コミには、**公開返信を慎重に行う**必要があります。感情的な反論は、炎上リスクがあります。

### テンプレート1: 差別的表現を指摘せず、事実を淡々と伝える

```
この度はご意見をいただき、ありがとうございます。

当店のスタッフは、国籍・性別・年齢を問わず、お客様に最高のサービスを提供できるよう日々努力しております。

もしサービスに不十分な点がございましたら、具体的にお教えいただけますと幸いです。

[店舗名] 店長
```

**ポイント**:
- 「差別的だ」と直接指摘しない（炎上リスクを避ける）
- 店の方針を淡々と伝える
- 他の閲覧者に「店は冷静に対応している」と印象づける

### テンプレート2: 人権尊重の姿勢を明確に示す

```
この度はご来店いただき、ありがとうございました。

当店では、すべてのお客様とスタッフの人権を尊重し、差別のない環境づくりに取り組んでおります。

ご意見をいただきましたが、当店の方針について何卒ご理解いただけますと幸いです。

[店舗名] 店長
```

**ポイント**:
- 「人権尊重」を明記
- 差別的発言には同調しない姿勢を示す
- 企業の社会的責任（CSR）をアピール

### テンプレート3: 返信しない（推奨）

**理由**:
- 差別的口コミに返信すると、議論が紛糾する
- 「差別主義者」と議論しても、建設的な結果にならない
- 他の閲覧者は、投稿内容を見れば「異常な投稿」と判断できる

**返信しない場合の対応**:
- 削除申請を粛々と行う
- 必要に応じて法的措置を検討

## 絶対に書いてはいけないNG返信表現

差別的口コミへの返信を誤ると、炎上リスクが高まります。

### NG例1: 感情的に反論する

```
NG返信: 「あなたの発言は差別です。許せません」
```

**リスク**:
- 感情的な返信は、逆に炎上を招く
- 「店側がヒステリックだ」と批判される

**正しい対応**:
- 淡々と店の方針を伝える

### NG例2: 差別主義者と議論する

```
NG返信: 「外国人スタッフも日本人と同じように働いています。差別しないでください」
```

**リスク**:
- 「差別しないでください」という表現は、上から目線と受け取られる
- 議論が長引き、他の閲覧者にも悪印象

**正しい対応**:
- 議論せず、削除申請

### NG例3: 逆差別的な表現を使う

```
NG返信: 「日本人だって悪いことをする人はいます」
```

**リスク**:
- 逆差別と受け取られ、さらに炎上

**正しい対応**:
- 特定の国籍・民族を引き合いに出さない

## 法的措置の検討

削除申請が却下された場合、法的措置を検討します。

### 民事訴訟（損害賠償請求）

**要件**:
- 差別的発言により、名誉が毀損された
- 売上減少などの具体的な損害が発生した

**損害賠償額の目安**:
- 個人: 10万円〜100万円
- 法人: 50万円〜300万円

**費用**:
- 弁護士費用: 50万円〜100万円
- 発信者情報開示請求: 50万円〜100万円

**期間**: 1年〜2年

### 刑事告訴（名誉毀損罪・侮辱罪）

**要件**:
- 差別的発言が、名誉毀損罪または侮辱罪に該当する

**手順**:
1. 警察に被害届を提出
2. 検察が起訴するか判断
3. 刑事裁判

**リスク**:
- 警察が「民事不介入」と判断し、受理されないことが多い
- 起訴されても、罰金刑で終わることが多い

### 人権擁護委員会への相談

**人権擁護委員会とは**:
- 法務省の外局である法務局に設置
- 人権侵害に関する相談を受け付け、調査・救済を行う

**相談方法**:
- 最寄りの法務局に相談
- 「みんなの人権110番」 0570-003-110（平日8:30〜17:15）

**人権擁護委員会ができること**:
- 相談者への助言
- 加害者への啓発・指導
- 関係機関への通告

**人権擁護委員会ができないこと**:
- 強制的な削除命令
- 刑事罰の科刑
- 損害賠償の命令

**効果**:
- 軽度の差別的発言には有効
- 加害者が反省すれば、削除に応じることも

## 予防策: 差別的口コミを受けにくくするために

差別的口コミを完全に防ぐことはできませんが、リスクを減らす対策はあります。

### 予防策1: 多様性尊重の方針を明示

ホームページやGoogleマイビジネスに、以下の方針を明記します。

**記載例**:
```
【当店の方針】

当店では、国籍・人種・性別・年齢・障害・宗教・性的指向を理由に、お客様やスタッフを差別することはありません。

すべての方に快適にご利用いただけるよう、スタッフ一同心がけております。
```

**効果**:
- 差別主義者が来店しにくくなる
- 「差別のない店」というブランディング

### 予防策2: スタッフ教育

差別的な発言を受けた場合の対応をスタッフに教育します。

**教育内容**:
- 差別的発言には反論しない（感情的にならない）
- 店長・本部に報告
- 必要に応じて警察・弁護士に相談

### 予防策3: 口コミ監視ツールの導入

差別的口コミは、早期発見・早期削除が重要です。

**推奨ツール**:
- Googleアラート（無料）
- レピュ研（月額2万円〜）
- クチコミル（月額1万円〜）

**効果**:
- 投稿されたらすぐに通知
- 削除申請が迅速にできる

## 差別的口コミの判例・事例

実際に削除申請・訴訟に発展した事例です。

### 事例1: 外国人経営者への人種差別口コミ（削除成功）

**概要**:
- 飲食店の経営者が中国人であることを理由に「不潔」「信用できない」と投稿
- Googleマップに削除申請
- 「人種差別的な内容」として削除

**教訓**: 明確な人種差別表現は、削除される。

### 事例2: LGBT差別口コミ（民事訴訟で賠償命令）

**概要**:
- 「ゲイの店員がいて気持ち悪い」と投稿
- 店側が発信者情報開示請求
- 投稿者に30万円の賠償命令

**教訓**: 性的指向への差別的発言は、損害賠償の対象。

### 事例3: 障害者差別口コミ（削除されず）

**概要**:
- 「車椅子の客がいて通路が狭く感じた」と投稿
- Googleマップに削除申請
- 「単なる感想」として削除されず

**教訓**: 「差別的意図」が明確でない場合、削除されにくい。
