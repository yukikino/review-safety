## この記事は誰のため？

この記事は、Googleマップや口コミサイトで「店内を無断撮影された」「スタッフの顔写真を勝手に投稿された」「個人情報を晒された」という被害を受けた店舗経営者・スタッフに向けて書かれています。

無断撮影・プライバシー侵害は、肖像権侵害・プライバシー権侵害として、民事訴訟の対象になります。また、悪質な場合は迷惑防止条例違反で刑事罰の可能性もあります。

この記事を読むことで、無断撮影された写真の削除申請方法、肖像権侵害の法的判断基準、損害賠償請求の流れが分かります。

## プライバシー侵害・肖像権侵害とは？

### 肖像権とは？

肖像権とは、自分の顔や姿を無断で撮影・公開されない権利です。

**法律上の根拠**:
- 憲法13条（幸福追求権）から派生
- 判例で確立された権利（明文規定はない）

**肖像権侵害の判断基準**（最高裁平成17年11月10日判決）:
1. 撮影・公開の必要性
2. 撮影方法・公開方法の相当性
3. 被撮影者の社会的地位
4. 撮影・公開による不利益の程度

**肖像権侵害になる例**:
- スタッフの顔写真を無断でGoogleマップに投稿
- 他の客の顔が写った店内写真を無断で投稿
- 店の外観に偶然写った通行人の顔を無修正で投稿

**肖像権侵害にならない例**:
- 店内の料理だけを撮影（人が写っていない）
- 顔にモザイクをかけて投稿
- 公道から見える店の外観を撮影（ただし、店内が写り込む場合は要注意）

### プライバシー権とは？

プライバシー権とは、私生活上の事柄をみだりに公開されない権利です。

**プライバシー侵害の判断基準**:
1. 私生活上の事実または私生活上の事実らしく受け取られるおそれのある事柄であること
2. 一般人の感受性を基準にして当該私人の立場に立った場合、公開を欲しないであろうと認められる事柄であること
3. 一般の人々に未だ知られていない事柄であること

**プライバシー侵害になる例**:
- スタッフの本名・住所・電話番号を投稿
- 経営者の家族構成・自宅住所を投稿
- 「店長は前科者だ」などの個人的情報を投稿（虚偽でも真実でも侵害になる）

## プライバシー侵害口コミの種類

### 1. 無断撮影された写真・動画

**具体例**:
- スタッフの顔写真を無断で投稿
- 他の客が写った店内写真を投稿
- 店内の様子を動画で撮影し、YouTubeにアップロード

**削除基準**:
- Googleマップ: 「プライバシーを侵害するコンテンツ」として削除対象
- 食べログ: 「個人のプライバシーを侵害する写真」は削除対象

### 2. 個人情報の公開

**具体例**:
- 「店長の名前は〇〇、住所は〇〇」と投稿
- 「スタッフの△△さんは、前職で〇〇していた」と投稿
- 経営者の家族の写真を投稿

**削除基準**:
- Googleマップ: 「個人情報を含むコンテンツ」として削除対象
- 個人情報保護法違反の可能性

### 3. 盗撮・隠し撮り

**具体例**:
- トイレ・更衣室を隠し撮り
- スカート内を盗撮
- 店内の防犯カメラ映像を無断で撮影

**法的評価**:
- 迷惑防止条例違反（各都道府県の条例により異なる）
- 盗撮の場合、刑事罰の対象（6ヶ月以下の懲役または50万円以下の罰金等）

### 4. なりすまし投稿

**具体例**:
- スタッフの名前を騙って口コミを投稿
- 「〇〇店のスタッフですが、この店はブラック企業です」と投稿

**法的評価**:
- 名誉毀損罪（刑法230条）
- 業務妨害罪（刑法233条）

## 緊急対応フローチャート

無断撮影・プライバシー侵害を受けた場合、以下の順序で対応します。

### ステップ1: 証拠保全（最優先）

**即座に実施すべきこと**:

1. **スクリーンショット撮影**
   - 投稿全文
   - 無断撮影された写真・動画
   - 投稿者のアカウント情報
   - 投稿日時
   - URLも含めて撮影

2. **ウェブ魚拓（archive.org等）に保存**
   - 削除申請前に必ず保存
   - 削除された後も証拠として残る

3. **被害者（スタッフ・客）への連絡**
   - 無断撮影された本人に連絡
   - 削除申請への協力を依頼

### ステップ2: プラットフォームへの削除申請

**Googleマップの場合**:

1. Google マップで該当の写真を表示
2. 右上の「︙」（三点リーダー）をクリック
3. 「写真を報告」を選択
4. 報告理由: 「プライバシーの侵害」を選択
5. 詳細を記載: 「写っている人物の許可を得ていない」等

**削除が認められやすいケース**:
- 人の顔が明確に写っている
- スタッフ・客の許可を得ていない
- トイレ・更衣室など、通常撮影が禁止されている場所

**削除が認められにくいケース**:
- 顔がぼやけて判別できない
- 公道から見える店の外観
- 料理だけの写真（人が写っていない）

**削除までの期間**: 通常3日〜1週間（緊急性が高い場合は早まることも）

**食べログの場合**:

1. 食べログの「口コミに関するお問い合わせ」フォームから申請
2. 該当写真のURL、削除理由を記載
3. 「個人のプライバシーを侵害する写真」として申請

**削除までの期間**: 通常1週間〜2週間

### ステップ3: 警察への相談（盗撮の場合）

**相談先**:
- 所轄の警察署（生活安全課）
- サイバー犯罪相談窓口（都道府県警察本部）

**持参すべき証拠**:
- スクリーンショット
- 投稿者のアカウント情報
- 被害状況をまとめたメモ

**警察が動くケース**:
- トイレ・更衣室の盗撮
- スカート内の盗撮
- 明らかに隠し撮りされた動画

**警察が動かないケース**:
- 単なる無断撮影（店内の様子等）
- 顔が写っているだけ
- 「民事不介入」と判断された場合

### ステップ4: 弁護士への相談

**相談すべきタイミング**:
- 削除申請が却下された場合
- 個人情報が大量に晒された場合
- 損害賠償請求を検討する場合

**弁護士の選び方**:
- インターネット法務に詳しい弁護士
- 肖像権侵害・プライバシー侵害の実績がある弁護士

**相談料の目安**:
- 初回相談: 30分5,000円〜1万円（無料相談もあり）
- 削除請求: 10万円〜30万円
- 発信者情報開示請求: 50万円〜100万円
- 損害賠償請求訴訟: 50万円〜150万円

## 公開返信テンプレート（プライバシー侵害口コミ用）

無断撮影・プライバシー侵害口コミには、**公開返信をしないことが原則**です。返信すると、投稿が削除されるリスクがあり、証拠が消えます。

### テンプレート1: 削除申請中であることを伝える

```
この度はご来店いただき、ありがとうございました。

投稿いただいた写真につきまして、写っている方のプライバシーに配慮し、削除申請をさせていただいております。

ご理解のほど、よろしくお願いいたします。

[店舗名] 店長
```

**ポイント**:
- 削除申請中であることを明記
- 他の閲覧者に「プライバシーに配慮している店」と印象づける

### テンプレート2: 撮影禁止を伝える

```
この度はご来店いただき、ありがとうございました。

当店では、他のお客様やスタッフのプライバシー保護のため、店内での撮影はご遠慮いただいております。

今後ともご協力のほど、よろしくお願いいたします。

[店舗名] 店長
```

**ポイント**:
- 店のルールを明記
- 「撮影禁止」を他の客にも周知

### テンプレート3: 返信しない（推奨）

**理由**:
- 返信すると、投稿者が写真を削除する可能性（証拠が消える）
- 削除申請を粛々と行う方が効果的

## 絶対に書いてはいけないNG返信表現

プライバシー侵害口コミへの返信を誤ると、炎上リスクがあります。

### NG例1: 投稿者を脅す

```
NG返信: 「無断撮影は犯罪です。警察に通報しました」
```

**リスク**:
- 投稿者が写真を削除し、証拠が消える
- 「店側が脅迫してきた」とSNSで拡散される

**正しい対応**:
- 静かに削除申請と警察相談

### NG例2: 被害者（スタッフ）の情報を公開返信に書く

```
NG返信: 「写っているスタッフの〇〇さんは、無断撮影されて困っています」
```

**リスク**:
- スタッフの名前を公開することで、さらにプライバシー侵害

**正しい対応**:
- 特定の個人名は出さない

### NG例3: 感情的に非難する

```
NG返信: 「盗撮は犯罪です。あなたは犯罪者です」
```

**リスク**:
- 名誉毀損で逆に訴えられる可能性

**正しい対応**:
- 淡々と削除申請

## 法的措置の検討

削除申請が却下された場合、法的措置を検討します。

### 削除仮処分申請

**手順**:
1. 弁護士に依頼
2. 裁判所に削除仮処分を申請
3. 裁判所がプラットフォーム（Google等）に削除命令

**費用**: 20万円〜50万円（弁護士費用含む）
**期間**: 1ヶ月〜3ヶ月

**削除が認められやすいケース**:
- 明らかな肖像権侵害
- 個人情報が晒されている
- 盗撮・隠し撮り

### 発信者情報開示請求

投稿者を特定し、損害賠償請求をする手続きです。

**手順**:
1. プラットフォーム（Google等）に仮処分申請
2. IPアドレス・タイムスタンプを取得
3. インターネットプロバイダ（ISP）に本開示請求
4. 投稿者の氏名・住所を特定
5. 損害賠償請求訴訟

**費用**: 50万円〜100万円（弁護士費用含む）
**期間**: 6ヶ月〜1年

### 損害賠償請求訴訟

**損害賠償額の目安**:
- 肖像権侵害: 10万円〜100万円
- プライバシー侵害: 50万円〜300万円
- 盗撮・隠し撮り: 100万円〜500万円

**判例**:
- スタッフの顔写真を無断投稿 → 30万円の賠償命令
- 経営者の自宅住所を晒した → 100万円の賠償命令
- トイレの盗撮動画を投稿 → 300万円の賠償命令

## 予防策: 無断撮影を防ぐために

無断撮影・プライバシー侵害を未然に防ぐための対策です。

### 予防策1: 店内に「撮影ルール」を掲示

**掲示例**:
```
【撮影に関するお願い】

当店では、他のお客様やスタッフのプライバシー保護のため、以下のルールをお守りください。

・料理のみの撮影は可能です
・他のお客様やスタッフが写らないようご配慮ください
・店内の様子を撮影する場合は、スタッフにお声がけください

ご協力のほど、よろしくお願いいたします。
```

**効果**:
- 無断撮影の抑止力
- トラブル時に「撮影ルール違反」として削除申請しやすい

### 予防策2: SNS投稿ガイドラインの作成

ホームページやGoogleマイビジネスに、SNS投稿のガイドラインを明記します。

**ガイドライン例**:
```
【SNS投稿ガイドライン】

当店では、お客様にSNSで当店の料理や雰囲気をご紹介いただくことを歓迎しております。

ただし、以下の点にご注意ください。

・他のお客様やスタッフの顔が写らないようにしてください
・写っている場合は、モザイク処理をお願いします
・店内の様子を動画で撮影する場合は、事前にスタッフにご確認ください

ご理解とご協力のほど、よろしくお願いいたします。
```

### 予防策3: スタッフへの教育

無断撮影を発見した場合の対応をスタッフに教育します。

**教育内容**:
- 「撮影はお控えください」と丁寧に伝える
- 強制的にカメラを取り上げない（暴行罪のリスク）
- 悪質な場合は店長に報告
- 盗撮の疑いがある場合は警察に通報

### 予防策4: 防犯カメラの設置

店内に防犯カメラを設置し、無断撮影の証拠を残します。

**注意点**:
- 防犯カメラの設置を店内に掲示（プライバシー配慮）
- 録画データの保管期間を定める（通常1ヶ月）
- 不正アクセスを防ぐためのセキュリティ対策

## プライバシー侵害の判例・事例

実際に削除申請・訴訟に発展した事例です。

### 事例1: スタッフの顔写真を無断投稿（削除成功）

**概要**:
- 飲食店のスタッフの顔写真を、Googleマップに無断投稿
- 店側がGoogleに削除申請
- 「プライバシー侵害」として削除

**教訓**: 人の顔が写った写真は、削除されやすい。

### 事例2: 経営者の自宅住所を晒した（損害賠償100万円）

**概要**:
- 口コミで「店長の自宅は〇〇町〇〇番地」と投稿
- 店側が発信者情報開示請求
- 投稿者に100万円の賠償命令

**教訓**: 個人情報の公開は、高額賠償のリスク。

### 事例3: トイレ盗撮動画をYouTubeに投稿（逮捕）

**概要**:
- 飲食店のトイレを隠し撮りし、YouTubeに投稿
- 店側が警察に被害届
- 投稿者が迷惑防止条例違反で逮捕

**教訓**: 盗撮は刑事罰の対象。警察は動く。
