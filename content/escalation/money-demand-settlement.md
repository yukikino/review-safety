## この記事は誰のため？

この記事は、Googleマップや口コミサイトで「慰謝料を払え」「示談金を用意しろ」「誠意を見せろ」などの金銭要求を受けた店舗経営者に向けて書かれています。

金銭要求には、正当な損害賠償請求と、不当な恐喝の2種類があります。対応を誤ると、恐喝に応じてしまったり、逆に正当なクレームを無視して訴訟リスクが高まったりします。

この記事を読むことで、正当な請求と不当な要求の見分け方、恐喝罪の判断基準、弁護士を通じた交渉方法、示談書の作成方法が分かります。

## 正当な損害賠償請求と不当な金銭要求の境界線

金銭要求を受けた場合、まず「正当な請求」か「不当な要求」かを見極めます。

### 正当な損害賠償請求

**定義**: 店側の過失により、実際に損害が発生し、その賠償を求める請求

**具体例**:

1. **食中毒による医療費請求**
   - 「店の料理で食中毒になり、病院代3万円かかった。払ってほしい」
   - **判断**: 因果関係が証明されれば、正当な請求

2. **商品の欠陥による損害**
   - 「買った服が破れていて、大事な会議で恥をかいた。クリーニング代5,000円を払え」
   - **判断**: クリーニング代は正当、「恥をかいた慰謝料」は過大請求の可能性

3. **予約ミスによる損害**
   - 「予約したのに席がなく、他の店を探した。タクシー代5,000円を払え」
   - **判断**: 実費（タクシー代）は正当、慰謝料は過大請求の可能性

### 不当な金銭要求（恐喝の可能性）

**定義**: 店側に過失がない、または過失があっても請求額が過大、または脅迫的な表現を伴う要求

**具体例**:

1. **根拠のない慰謝料請求**
   - 「料理がまずかった。精神的苦痛を受けたので慰謝料10万円払え」
   - **判断**: 「まずい」は主観的評価。慰謝料請求の根拠なし

2. **過大な金額請求**
   - 「服にソースがついた。クリーニング代として5万円払え」（実際のクリーニング代は2,000円程度）
   - **判断**: 過大請求。恐喝の可能性

3. **脅迫を伴う金銭要求**
   - 「5万円払わないと、ネットで拡散する」
   - 「誠意を見せないと、保健所に通報する」
   - **判断**: 恐喝罪（刑法249条）に該当する可能性

## 恐喝罪の成立要件

金銭要求が恐喝罪に該当するかの判断基準です。

### 恐喝罪（刑法249条）

**定義**: 人を恐喝して財物を交付させること、または財産上不法の利益を得ること

**成立要件**:
1. 恐喝行為（脅迫または暴行）
2. 相手方の畏怖（恐怖を感じること）
3. 財物の交付または財産上の利益

**法定刑**: 10年以下の懲役

### 恐喝罪に該当する表現

**具体例**:
- 「5万円払わないと、ネットで拡散する」
- 「誠意を見せないと、店を潰す」
- 「金を払えば、低評価レビューを消してやる」
- 「示談金を払わないと、警察に行く」（正当な理由がない場合）
- 「慰謝料を払わないと、マスコミに通報する」

**ポイント**:
- 「〇〇しないと△△する」という条件付きの脅迫
- 「誠意を見せろ」「分かっているだろう」などの暗示的な金銭要求

### 恐喝罪に該当しない表現

**具体例**:
- 「医療費3万円を払ってください」（具体的な損害の請求）
- 「弁護士に相談します」（正当な権利行使の予告）
- 「消費者センターに相談します」（正当な権利行使の予告）

**ポイント**:
- 具体的な損害額の請求
- 正当な権利行使（弁護士相談、消費者センター相談等）の予告

## 金銭要求を受けた場合の対応フローチャート

### ステップ1: 証拠保全（最優先）

**即座に実施すべきこと**:

1. **スクリーンショット撮影**
   - 口コミ全文
   - DM・メールのやり取り
   - 投稿者のアカウント情報
   - 投稿日時

2. **音声・動画の録音（電話・対面の場合）**
   - スマホの録音アプリで録音
   - 「録音しています」と事前に伝える必要はない（一方当事者の同意があれば合法）

3. **やり取りの記録**
   - いつ、どこで、誰から、どのような要求があったか
   - 要求された金額
   - 要求の理由

### ステップ2: 正当性の判断

**チェックリスト**:

| 項目 | 正当な請求 | 不当な要求 |
|------|----------|----------|
| 損害の具体性 | 医療費、交通費等の実費 | 「精神的苦痛」など曖昧 |
| 因果関係 | 店の過失と損害の因果関係が明確 | 因果関係が不明確 |
| 金額の妥当性 | 実費相当 | 実費に対して過大 |
| 要求の仕方 | 「請求します」 | 「払わないと〇〇する」 |
| 証拠の有無 | 領収書、診断書等あり | 証拠なし |

**判断に迷う場合**:
- 弁護士に相談（初回相談30分5,000円〜1万円）
- 消費者センターに相談（無料）

### ステップ3: 対応方法の選択

**正当な請求の場合**:
1. 謝罪
2. 損害額の確認（領収書、診断書等の提出を依頼）
3. 弁護士を通じて示談交渉
4. 示談書の作成

**不当な要求の場合**:
1. 一切の金銭支払いを拒否
2. 警察に相談（恐喝罪の可能性）
3. 弁護士に相談（削除請求・発信者情報開示請求）
4. プラットフォームに削除申請

### ステップ4: 弁護士を通じた交渉

**弁護士に依頼すべき理由**:
- 法的根拠のない要求を拒否できる
- 示談金額の妥当性を判断できる
- 示談書の作成を依頼できる
- 店側が直接交渉するリスクを避けられる

**弁護士費用の目安**:
- 初回相談: 30分5,000円〜1万円
- 示談交渉: 10万円〜30万円
- 訴訟対応: 50万円〜150万円

## 絶対にやってはいけないNG対応

金銭要求への対応を誤ると、恐喝に応じたり、逆に訴訟リスクが高まったりします。

### NG対応1: 安易に金銭を支払う

```
相手の要求: 「慰謝料10万円払え」
NG対応: 「分かりました。すぐにお支払いします」
```

**リスク**:
- 恐喝に応じると、さらに要求がエスカレートする
- 支払った事実が証拠となり、「店が非を認めた」と主張される
- 他のクレーマーを呼び込むリスク

**正しい対応**:
- 「弁護士と相談します」と伝え、支払いを保留
- 損害の具体的内容を確認（領収書等の提出を依頼）

### NG対応2: 口頭での約束

```
相手の要求: 「5万円払えば、レビューを消す」
NG対応: 「分かりました。後日お支払いします」
```

**リスク**:
- 口頭での約束は、後で「言った・言わない」のトラブルになる
- 「5万円払ったのに、レビューを消してくれない」とさらに揉める

**正しい対応**:
- 示談する場合は、必ず書面（示談書）を作成
- 弁護士を通じて交渉

### NG対応3: 感情的に反論する

```
相手の要求: 「誠意を見せろ」
NG対応: 「ふざけるな。あなたの方こそクレーマーだ」
```

**リスク**:
- 感情的な対応は、逆に名誉毀損で訴えられる
- やり取りが録音されていた場合、不利な証拠になる

**正しい対応**:
- 冷静に「弁護士と相談します」と伝える

### NG対応4: 相手の要求を全て飲む

```
相手の要求: 「謝罪文をSNSに投稿しろ。慰謝料10万円払え。社長が直接謝罪に来い」
NG対応: 「全て対応します」
```

**リスク**:
- 一度要求を飲むと、さらにエスカレートする
- 謝罪文のSNS投稿は、逆に炎上リスク

**正しい対応**:
- 「弁護士と相談します」と伝え、一旦保留

## 公開返信テンプレート（金銭要求口コミ用）

金銭要求を含む口コミには、**公開返信をしないことが原則**です。返信すると、投稿者が削除するリスクがあります。

### テンプレート1: 弁護士に相談済み

```
この度はご連絡いただき、ありがとうございます。

お客様からのご要望につきまして、弁護士と相談の上、適切に対応させていただきます。

恐れ入りますが、今後のやり取りは弁護士を通じて行わせていただきます。

[店舗名] 店長
```

**ポイント**:
- 「弁護士」という言葉で、法的対応を示唆
- 直接のやり取りを拒否

### テンプレート2: 事実確認を依頼

```
この度はご連絡いただき、ありがとうございます。

お客様のご請求内容につきまして、詳しくお話をお伺いしたく、恐れ入りますが店舗まで直接ご連絡いただけますでしょうか。

領収書や診断書等の資料をご提示いただけますと幸いです。

[店舗名] 店長
```

**ポイント**:
- 証拠の提示を求める
- 公開の場では金額の話をしない

### テンプレート3: 返信しない（最も推奨）

**理由**:
- 公開返信すると、投稿者が削除する可能性（証拠が消える）
- 弁護士を通じて、非公開で交渉する方が効果的

## 示談交渉の進め方

正当な請求の場合、示談交渉を行います。

### 示談交渉の流れ

**ステップ1: 損害額の確認**

**依頼すべき資料**:
- 医療費の領収書
- 診断書
- 交通費の領収書
- その他実費の証明

**ステップ2: 妥当な示談金額の算定**

**示談金の内訳**:
- 実費（医療費、交通費等）
- 慰謝料（実費の1〜3割程度が目安）

**具体例**:
- 食中毒で医療費3万円かかった場合
  - 実費: 3万円
  - 慰謝料: 1万円〜3万円
  - 合計: 4万円〜6万円

**ステップ3: 示談書の作成**

示談書には、以下の条項を必ず含めます。

**示談書の必須条項**:
1. 示談金額
2. 支払い方法・期日
3. 清算条項（「本件に関し、今後一切の請求をしない」）
4. 口コミ削除の約束（該当する場合）
5. 秘密保持条項（「本件について、第三者に口外しない」）

**示談書の例**:
```
示談書

〇〇株式会社（以下「甲」という）と、△△（以下「乙」という）は、〇年〇月〇日に甲の店舗で発生した事故について、以下のとおり示談する。

第1条（示談金）
甲は乙に対し、示談金として金〇万円を支払う。

第2条（支払方法）
甲は、本示談書締結日より7日以内に、乙の指定する銀行口座に振り込む方法により支払う。

第3条（清算条項）
甲及び乙は、本件に関し、本示談書に定めるもののほか、今後一切の債権債務がないことを相互に確認する。

第4条（口コミ削除）
乙は、Googleマップに投稿した口コミを、本示談書締結後速やかに削除する。

第5条（秘密保持）
甲及び乙は、本件について、第三者に口外しないことを約束する。

〇年〇月〇日

甲: 〇〇株式会社 代表取締役 〇〇
乙: △△（署名・捺印）
```

**ステップ4: 示談金の支払い**

**支払い方法**:
- 銀行振込（振込明細を保管）
- 現金手渡し（必ず領収書をもらう）

**注意点**:
- 示談書を交わす前に支払わない
- 口コミ削除を確認してから支払う（または同時履行）

## 予防策: 不当な金銭要求を受けにくくするために

### 予防策1: クレーム対応マニュアルの整備

**マニュアルに含めるべき内容**:
- クレームの種類別対応方法
- 金銭要求を受けた場合のエスカレーションルール
- 絶対に言ってはいけないNG表現（「お詫びにお金をお渡しします」等）

### 予防策2: スタッフ教育

**教育内容**:
- 「お詫びに〇〇します」と安易に約束しない
- 金銭要求を受けたら、即座に店長・本部に報告
- 感情的にならず、冷静に対応

### 予防策3: 録音・録画の準備

クレーム対応時は、必ず録音・録画します。

**録音・録画の方法**:
- スマホの録音アプリ
- 店内の防犯カメラ
- ボイスレコーダー

**法的根拠**:
- 一方当事者（店側）の同意があれば、録音は合法
- 「録音しています」と事前に伝える必要はない

### 予防策4: 顧問弁護士との契約

金銭要求トラブルは、初動が重要です。

**顧問契約のメリット**:
- すぐに相談できる
- 示談交渉を依頼できる
- 抑止力になる（「顧問弁護士がいる」と明記）

**費用**: 月額3万円〜5万円

## 金銭要求トラブルの判例・事例

### 事例1: 食中毒クレームで100万円要求（恐喝未遂で逮捕）

**概要**:
- 「食中毒になった。慰謝料100万円払え。払わないとネットで拡散する」と脅迫
- 店側が警察に被害届
- 投稿者が恐喝未遂罪で逮捕

**判決**: 懲役1年（執行猶予3年）

**教訓**: 過大な金額要求 + 脅迫 = 恐喝罪

### 事例2: 衣服汚損で正当な賠償請求（示談成立）

**概要**:
- 店員がワインをこぼし、客の服を汚した
- クリーニング代5,000円 + 慰謝料1万円を請求
- 店側が謝罪し、示談金1.5万円で示談成立

**教訓**: 正当な請求には、誠実に対応する。

### 事例3: 「誠意を見せろ」暗示的要求（示談拒否・訴訟）

**概要**:
- 「誠意を見せろ」と曖昧な金銭要求
- 店側が「具体的な損害額を教えてください」と返答
- 相手が「分かっているだろう」と回答
- 店側が示談を拒否し、相手が提訴
- 裁判所が「具体的な損害の立証なし」として請求棄却

**教訓**: 曖昧な要求には応じない。具体的な損害の立証を求める。
