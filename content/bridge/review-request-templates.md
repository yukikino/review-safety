口コミ依頼は「何を言うか」と同じくらい「どの場面で・どう伝えるか」が重要です。この記事では、メール・SMS・LINE・店頭POP・会計時の声かけという5つの場面別に、すぐに使えるテンプレート15選を紹介します。

## 口コミ依頼の基本ルール

テンプレートを使い始める前に、Googleガイドライン違反にならないための基本ルールを確認しておきましょう。口コミ依頼の詳しいルールは[Googleマップ口コミを増やす方法](/playbook/review-generation)でも解説しています。

### 守るべき3原則

**原則1: 評価を指定しない**

「★5をお願いします」は明確なガイドライン違反です。「ご感想をお聞かせください」という中立的な表現を必ず使いましょう。

**原則2: 報酬と紐づけない**

「レビュー投稿で10%オフ」「口コミを書いたら粗品プレゼント」はNGです。特典がある場合でも、レビュー依頼とは別のセクションに記載し、明確に分離してください。

**原則3: 実際の顧客にだけ依頼する**

来店・利用していない人にレビューを依頼することは虚偽のレビュー投稿に該当します。友人や知人であっても、実際にサービスを利用した場合にのみ依頼しましょう。

### NG表現チェックリスト

以下の表現が含まれていたら、テンプレートを修正してください。

- 「★5レビューをお願いします」→ 評価指定はNG
- 「良い口コミを書いてください」→ 高評価誘導はNG
- 「レビュー投稿でクーポンプレゼント」→ 報酬紐づけはNG
- 「もしご不満があれば先に店舗にご連絡ください」→ 低評価抑制はNG

## 場面別テンプレート15選

### 会計時の声かけテンプレート（テンプレート1〜3）

会計時はお客様の満足度が高い状態であり、口コミ依頼の成功率が最も高いタイミングです。

**テンプレート1: 基本形（全業種対応）**
```
本日はありがとうございました。もしよろしければ、
Googleマップでご感想をお聞かせいただけると嬉しいです。
こちらのQRコードから簡単に投稿できます。
```

**テンプレート2: リピーター向け**
```
いつもご利用ありがとうございます。もしお時間がありましたら、
Googleマップで当店のことをシェアしていただけると
スタッフ一同の励みになります。
```

**テンプレート3: 感動の瞬間に寄り添う形**
```
お気に召していただけて本当に嬉しいです。
もしよろしければ、その体験をGoogleマップで
教えていただけませんか？今後のサービスの参考にもなります。
```

**声かけのポイント**: 笑顔で、押し付けがましくならないトーンで伝えます。断られたら「ありがとうございました」と笑顔で送り出し、それ以上は追いかけないことが大切です。

### メールテンプレート（テンプレート4〜6）

来店後24時間以内に送信するのが最も効果的です。件名でお礼を伝え、本文の最後にレビューリンクを添えるのが自然な流れです。

**テンプレート4: 飲食店向けメール**
```

お食事はいかがでしたでしょうか？

当店では、お客様のご意見をもとにメニューやサービスの
改善に取り組んでおります。もしお時間がございましたら、
Googleマップでご感想をお聞かせいただけますと幸いです。

▼こちらから投稿できます（所要時間：約1分）
[レビューリンク]

またのご来店を心よりお待ちしております。

[店舗名]
[住所・電話番号]
```

**テンプレート5: 美容室・サロン向けメール**

## 業種別カスタマイズのポイント

テンプレートはそのままでも使えますが、業種に合わせてカスタマイズすることで反応率が上がります。

### 飲食店の場合

飲食店では「お食事」「メニュー」「お料理」といった具体的なワードを入れると、お客様が「何について書けばいいか」が明確になり、レビューを書きやすくなります。

カスタマイズ例として、「もしよろしければ、本日のお料理のご感想をGoogleマップでお聞かせください」のように、具体的な体験に触れる表現が効果的です。

送信タイミングは来店後12〜24時間以内が最適です。食事の記憶が鮮明なうちにお送りしましょう。

### 美容室・サロンの場合

美容室では「仕上がり」「スタイル」「施術」といったワードが効果的です。また、担当スタイリスト名を入れることで個人的なつながりを感じてもらえます。

カスタマイズ例として、「担当の[スタイリスト名]です。本日の仕上がりにご満足いただけましたら、Googleマップでご感想をシェアしていただけると嬉しいです」のように、担当者からの個人的なメッセージにすると反応率が上がります。

### クリニック・歯科医院の場合

医療機関では、まずアフターケア情報を伝えた上で、最後にレビュー依頼を添える構成が効果的です。患者さんの健康を第一に考えている姿勢を示すことで、信頼感が高まります。

「施術後のケアについてご不明な点がございましたら、いつでもお問い合わせください」というアフターケア案内を先に置き、その後に「また、当院へのご意見もお待ちしております」と自然につなげます。

### 小売店・EC連動の場合

小売店では「購入した商品」に焦点を当てると効果的です。商品使用後のフィードバックという切り口で依頼すれば、お客様も書きやすく感じます。

購入後3〜7日程度が最適な送信タイミングです。商品を実際に使い始めたタイミングで依頼することで、具体的な体験に基づいたレビューを書いてもらえます。

## テンプレート運用のコツ

### 反応率を上げる3つの工夫

**工夫1: 所要時間を明記する**

「所要時間：約1分」と書くだけで、「面倒そう」という心理的ハードルが下がります。実際にGoogleマップのレビュー投稿は1〜2分で完了するため、正確な情報です。

**工夫2: 複数チャネルを組み合わせる**

会計時の声かけだけでなく、24時間以内のメール・LINEフォローアップを組み合わせることで、口コミ投稿率は平均2〜3倍に向上します。声かけで意識してもらい、メール・LINEで行動を促すという2段階のアプローチです。

**工夫3: 月ごとにテンプレートを見直す**

同じテンプレートを半年以上使い続けると、リピーターは「またこのメールか」と感じてしまいます。季節の挨拶や新メニューの情報を盛り込みながら、定期的に文面を更新しましょう。

### スタッフへの共有方法

テンプレートをスタッフ全員で共有する際は、以下の3点を伝えてください。

1. **なぜ口コミが大切か**: 口コミ数と評価がGoogleマップの検索順位に直結し、集客に大きく影響すること
2. **NG表現の具体例**: 「★5をお願いします」が絶対にNGであること
3. **断られた時の対応**: 笑顔で「ありがとうございました」と送り出し、気まずい空気を作らないこと

## よくある質問

### Q1: テンプレートをそのまま使っても問題ありませんか？

**A**: はい、本記事のテンプレートはGoogleビジネスプロフィールのポリシーに準拠した内容で作成しています。ただし、店舗名や担当者名、商品名などの固有情報は必ず差し替えてください。また、自店の雰囲気やお客様との関係性に合わせて、トーンを調整することでより自然な依頼になります。

### Q2: メールとLINE、どちらで依頼した方が反応率が高いですか？

**A**: 一般的にLINEの方が開封率が高く（メールの開封率20〜30%に対しLINEは60〜80%）、反応率も高い傾向にあります。ただし、クリニックや高級店など、フォーマルなコミュニケーションが求められる業態ではメールの方が適しています。顧客層に合わせて使い分けるか、両方を併用するのが最も効果的です。

### Q3: 口コミ依頼の頻度はどのくらいが適切ですか？

**A**: 同一のお客様に対しては、1回の来店につき1回の依頼が原則です。来店時の声かけ+24時間以内のメールまたはLINE送信の計2タッチまでが適切な頻度です。3回以上の依頼はしつこいと感じられ、逆効果になります。リピーターには2〜3回目の来店時に再度依頼するのが良いタイミングです。
