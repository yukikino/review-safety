## この記事は誰のため？

この記事は、「お気に入りのお店に良い口コミを書きたいけれど、何をどう書けばいいか分からない」「お店に喜んでもらえるレビューを投稿したい」「他の人の参考になる口コミを書きたい」という消費者の方に向けて書かれています。

口コミは、お店にとって貴重なフィードバックであり、集客にも大きな影響を与えます。しかし、「美味しかったです」「良かったです」だけの一言レビューでは、お店側も他のユーザーも情報が不足して困ってしまいます。

この記事を読むことで、お店に喜ばれ、他のユーザーの参考にもなる口コミの書き方のコツと、業種別の具体的な例文が分かります。

## お店に喜ばれる口コミの5つのポイント

### ポイント1: 具体的なメニュー・サービス名を書く

お店にとって最も嬉しいのは、具体的な商品やサービスに言及してもらえることです。「料理が美味しかった」よりも「石窯で焼いたマルゲリータが絶品だった」と書いてもらえた方が、何が評価されているかが明確になります。

具体名を挙げることで、他のユーザーが注文の参考にすることもでき、口コミの価値が飛躍的に高まります。

**良い例:** 「ランチの日替わりパスタ（本日はボンゴレビアンコ）が特に美味しかったです」

**惜しい例:** 「パスタが美味しかったです」

### ポイント2: スタッフの対応に触れる

接客やサービスの質に言及することは、お店のスタッフにとって大きな励みになります。特に名前を出して褒めてもらえると、そのスタッフのモチベーション向上につながります。

**良い例:** 「担当の山田さんが髪質の悩みを丁寧にヒアリングしてくれて、自分に合ったスタイルを提案してくれました」

### ポイント3: 利用シーンを明記する

「誰と」「どんな目的で」利用したかを書くと、同じシチュエーションで店を探している人の参考になります。

**良い例:** 「妻の誕生日ディナーで利用しました。個室を用意していただき、サプライズのデザートプレートもお願いしたのですが、スタッフの方が素敵に演出してくれました」

### ポイント4: 写真を添付する

百聞は一見にしかず。料理の写真、店内の雰囲気、商品の仕上がりなどの写真があると、口コミの信頼性と参考度が格段に上がります。

写真を撮影する際のポイントとして、料理は真上や斜め45度からが見栄え良く撮れます。店内は全体の雰囲気が分かるように広角で撮ると良いでしょう。ただし、他のお客様が写り込まないよう注意してください。

### ポイント5: 改善点も率直に書く

お店に喜ばれる口コミは、褒めるだけの口コミではありません。改善点を丁寧な言葉で率直に伝えることも、お店にとっては非常に価値があります。ただし、攻撃的にならず建設的な表現を心がけましょう。

**良い例:** 「料理は大変美味しかったのですが、ランチタイムは混雑していて15分ほど待ちました。予約できるとなお良いと思います」

**NGな例:** 「待ち時間が長すぎてありえない。二度と行かない」

## 業種別・良い口コミ例文集

### 飲食店（レストラン・カフェ）

**例文1: イタリアンレストラン**
```
友人4人でディナーに利用しました。
前菜の生ハムとブッラータチーズの盛り合わせが特に美味しく、
パスタは「自家製タリアテッレのボロネーゼ」を注文しました。
手打ち麺のもちもちした食感と濃厚なソースが絶妙でした。

ワインはスタッフの方がおすすめしてくれたトスカーナの赤が
料理とよく合いました。ワインの知識が豊富で頼もしかったです。

店内は落ち着いた雰囲気で、友人との会話も楽しめました。
4人で合計2万円ほど。このクオリティでこの価格はかなりお得だと思います。
次回は石窯ピザも試してみたいです。
```

**例文2: ラーメン店**
```
平日12時過ぎに訪問。10分ほど並んで入店。
特製味噌ラーメン（950円）を注文しました。

スープは味噌のコクがしっかりありながらも後味がすっきりしていて、
最後まで飲み干せました。麺は中太の縮れ麺で、スープがよく絡みます。
チャーシューは厚切りで柔らかく、3枚入っているのが嬉しいです。

カウンター席のみの小さなお店ですが、清潔感があります。
店主の方が一人で切り盛りしており、提供も早かったです。
味噌ラーメン好きの方には強くおすすめします。
```

**口コミに含めると良い情報（飲食店）:**
- 注文したメニュー名と価格帯
- 味の感想（具体的に）
- 店内の雰囲気（広さ、清潔感、BGM等）
- 混雑状況と待ち時間
- 接客の印象
- 利用人数と合計金額の目安

### 美容院・美容室

**例文1: カット＆カラー**
```
担当の佐藤さんにカットとカラーをお願いしました。
「伸ばしかけだけど重くならないようにしたい」という
ざっくりした要望にも関わらず、丁寧にヒアリングしてくれて、
レイヤーを入れた軽やかなスタイルを提案してくれました。

カラーはアッシュベージュにしたのですが、
色持ちが良く、2週間経った今でもきれいな色味です。
ブリーチなしでこの透明感が出るのは技術力が高い証拠だと思います。

店内はおしゃれで居心地が良く、
ドリンクサービスも嬉しいポイントでした。
カット+カラーで12,000円。次回もお願いする予定です。
```

**口コミに含めると良い情報（美容院）:**
- 担当スタッフ名
- 施術内容と要望
- 仕上がりの感想（当日の感想だけでなく数日後の持ちも）
- 店内の雰囲気
- 料金

### クリニック（歯科・美容・内科等）

**例文1: 歯科クリニック**
```
虫歯治療で通院しました。院長先生が治療前に
レントゲン画像を見せながら丁寧に説明してくれたので、
治療内容と費用に納得してから進められました。

麻酔の注射がほとんど痛くなく驚きました。
表面麻酔をしてから注射するそうで、
歯医者が苦手な方にもおすすめできます。

受付の方も感じが良く、予約変更の電話にも
快く対応してくれました。
待ち時間は予約通りほぼなしでした。
駐車場が3台分あるので車でも通いやすいです。
```

**口コミに含めると良い情報（クリニック）:**
- 受診科目・症状の概要
- 診察・治療の丁寧さ
- 説明の分かりやすさ
- 待ち時間
- スタッフの対応
- アクセスの良さ

**注意:** クリニックの口コミでは、治療効果を断定的に書くことは避けましょう。「私の場合は○○でした」という個人の体験として書くのが適切です。

### ホテル・旅館

**例文1: ビジネスホテル**
```
出張で2泊利用しました。
駅から徒歩3分の立地が便利でした。

部屋はシングルルームでしたが、デスクが広く
ノートPCを広げて仕事するのに十分なスペースがありました。
Wi-Fiも安定しており、オンライン会議も問題なくできました。

朝食バイキングは種類が豊富で、特にだし巻き卵と
地元野菜のサラダが美味しかったです。

一つ気になったのは、隣の部屋の音が少し聞こえたことです。
耳栓があると安心かもしれません。

チェックイン時のフロントスタッフの対応が丁寧で
好印象でした。出張での宿泊先として、またリピートします。
1泊朝食付き8,500円でした。
```

**口コミに含めると良い情報（ホテル）:**
- 宿泊目的（出張、旅行、記念日等）
- 部屋タイプと広さの印象
- 設備（Wi-Fi、デスク、アメニティ等）
- 朝食の内容と感想
- アクセスの良さ
- 料金

## 避けるべきNGな書き方

良い口コミを書こうとするあまり、以下のようなNGパターンに陥らないよう注意しましょう。

### NG1: 個人情報を記載する

**やってはいけないこと:**
- スタッフのフルネームを許可なく記載する
- 他のお客様の特徴や会話内容を書く
- 自分の詳細な個人情報（住所、電話番号等）を含める

**対策:** スタッフ名は「担当の○○さん」のようにファーストネームまでにするか、「担当のスタイリストさん」のように役職で言及しましょう。

### NG2: 事実と異なることを書く

**やってはいけないこと:**
- 行っていない店の口コミを書く
- 食べていないメニューの感想を書く
- 友人に頼まれて代わりに書く

口コミは自分自身が実際に体験したことに基づいて書くものです。虚偽の口コミはプラットフォームのガイドライン違反であり、削除対象になります。

### NG3: 感情的に攻撃する

**やってはいけないこと:**
- 店名やスタッフ名を挙げて罵倒する
- 「潰れろ」「最低最悪」等の攻撃的な表現を使う
- 脅迫めいた書き方をする（「改善しなければ拡散する」等）

不満がある場合も、建設的な表現を心がけましょう。お店側も人間です。

### NG4: 情報量が少なすぎる

**惜しい口コミの例:**
- 「良かったです」
- 「普通」
- 「★5」（コメントなし）

短すぎる口コミは、他のユーザーの参考にならず、お店側も何が良かったのか分かりません。最低でも3〜5行は書くことを心がけましょう。

### NG5: 競合店の宣伝をする

**やってはいけないこと:**
- 「隣の○○店の方が美味しい」
- 「この値段なら××店に行った方がいい」

口コミは訪問した店舗についての感想を書く場所であり、他店の宣伝や比較は適切ではありません。

## 口コミを書くときの心構え

### 「未来のお客様」に向けて書く

口コミは、あなたと同じようにお店を探している「未来のお客様」が読みます。あなたがお店を選ぶとき、どんな情報があれば助かるかを考えながら書くと、自然と良い口コミになります。

### 「お店への手紙」として書く

口コミはお店へのフィードバックでもあります。良かった点は具体的に褒め、改善してほしい点は丁寧な言葉で伝えましょう。あなたの一つの口コミが、お店のサービス改善につながることもあります。

### 時間が経ってからでも書いてOK

「行ったけど書きそびれた」という場合でも、後から書いて大丈夫です。むしろ、少し時間が経ってから冷静に書くことで、より客観的で役立つ口コミになることもあります。ただし、記憶が新しいうちに書いた方が具体的な情報を盛り込めるため、可能であれば1週間以内に書くのがおすすめです。

口コミ対応の全体像を知りたい方は[口コミ対応完全ガイド](/playbook/complete-workflow)もご参照ください。

## よくある質問

### Q1: 星の数（★評価）はどうやって決めればいいですか？

**A**: 一般的な基準として、★5は「期待を大きく上回った、人におすすめしたい」、★4は「満足した、また行きたい」、★3は「普通、特に不満はないが感動もない」、★2は「期待を下回った、改善してほしい点がある」、★1は「非常に不満、大きな問題があった」です。完璧でなくても満足できたなら★4以上をつけるのが一般的で、★3は店舗側にとってはネガティブ寄りの評価として受け止められることが多いです。

### Q2: 口コミは匿名で書いた方がいいですか？実名で書いた方がいいですか？

**A**: Googleマップの場合、Googleアカウント名が表示されます。本名でもニックネームでも問題ありませんが、口コミの信頼性という観点では、プロフィール写真を設定し、ある程度の情報が見えるアカウントの方が信頼されやすい傾向があります。ただし、個人を特定されたくない場合はニックネームで全く問題ありません。

### Q3: 一度投稿した口コミは修正・削除できますか？

**A**: はい、Googleマップの口コミは投稿後にいつでも修正・削除できます。「後から思い出した情報を追記したい」「お店が改善してくれたので評価を上げたい」といった場合は、遠慮なく修正してください。口コミの修正は、お店にとっても嬉しいフィードバックになります。
